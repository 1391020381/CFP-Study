### 问题 3: 财务三表是哪三张？
**学生的回答**: 资产负债表、现金流量表、利润表

**初始理解**: 正确！学生知道三表的名称

**给出的解释**: 用个人生活类比解释三张表：
- 资产负债表 = 你有多少钱
- 利润表 = 你赚了多少钱
- 现金流量表 = 你兜里有多少现金

**理解检查**: ✅ 完全正确

### 问题 4: 为什么有了利润表还需要现金流量表？
**学生的回答**: 现金可以需要做其他事情，例如支付应付款、短期借债等

**初始理解**: 学生理解了现金的用途，但对"利润≠现金"的核心概念还需加强

**给出的解释**: 用两个场景说明
- 场景1：卖出商品但客户赊账 → 有利润但没现金
- 场景2：预付房租等支出 → 有现金流出但利润表体现较少

**核心概念**:
- 权责发生制（利润表）vs 收付实现制（现金流量表）
- 利润可以造假，现金流更难造假

**理解检查**: ✅ 学生理解了权责发生制和收付实现制的区别

### 问题 5: 茅台赊销案例
**场景**: 茅台卖出1000瓶酒，经销商3个月后付款，三张表如何体现？

**学生的回答**:
- 利润表：1000瓶酒的净利润 ✅
- 现金流量表：0 ✅
- 资产负债表：1000瓶酒的净利润 ⚠️ 需要调整

**给出的解释**:
- 资产负债表应体现为"应收账款"（资产）+ "留存收益"（权益）
- 三表勾稽关系：利润表净利润 → 资产负债表留存收益 → 现金流量表经营现金流

**理解检查**: ✅ 学生理解了三表的勾稽关系

### 问题 6: 实战判断
**场景**: 公司利润表赚1亿，但经营现金流为负，意味着什么？
A. 公司赚了很多钱
B. 公司可能赊销太多
C. 公司在造假

**学生的回答**: B ✅

**初始理解**: 学生理解了利润质量的问题

**给出的解释**: 净利润现金比率 = 经营现金流 ÷ 净利润
- >1：质量好
- ≈1：正常
- <1：要注意
- <0：危险信号

**理解检查**: ✅ 学生掌握了判断利润质量的实用指标

## 涵盖的关键概念

- **三大财务报表**: 资产负债表、利润表、现金流量表
- **权责发生制**: 交易发生时就记账（利润表）
- **收付实现制**: 实际收付现金时才记账（现金流量表）
- **应收账款**: 赊销产生的债权，是利润和现金的桥梁
- **留存收益**: 未分配的净利润，归属于股东
- **净利润现金比率**: 经营现金流 ÷ 净利润，判断利润质量
- **勾稽关系**: 三张表之间的逻辑联系
- **毛利率**: 毛利 ÷ 营业收入，衡量定价能力
- **净利率**: 净利润 ÷ 营业收入，衡量最终盈利能力
- **纸面富贵**: 利润表显示盈利但现金流很差的情况

## 快速判断框架（已掌握）

```
第一步：看毛利率
  ├─ >50% → 优秀（有定价权）
  ├─ 20%-50% → 一般
  └─ <20% → 要小心

第二步：看净利率
  ├─ >20% → 很好
  ├─ 10%-20% → 正常
  └─ <10% → 微利

第三步：看净利润现金比率（最重要！）
  ├─ >1 → 优秀 ✅
  ├─ ≈1 → 健康
  └─ <1 → 有问题 ⚠️
```
