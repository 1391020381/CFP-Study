# 上海证券交易所汇总数据详解

> 最后更新：2026-02-04
> 数据来源：akshare `stock_sse_summary()` 接口

---

## 数据列说明

| 列名 | 说明 |
|------|------|
| **股票** | 全部市场合计（主板 + 科创板） |
| **主板** | 沪市主板股票 |
| **科创板** | 科创板（STAR Market）股票，专注于科技创新企业 |

---

## 各项指标详细解释

| 项目 | 含义 | 单位 | 说明 |
|------|------|------|------|
| **流通股本** | 可自由交易的股票数量 | 亿股 | 可以在二级市场买卖的股份，不包括限售股 |
| **总市值** | 所有股票的市场价值总和 | 亿元 | 计算公式：股价 × 总股本，反映市场整体规模 |
| **平均市盈率** | 市盈率的加权平均值 | 倍 | 股价/每股收益的比率，越高表示估值越高 |
| **上市公司** | 上市公司的总数 | 家 | 在该交易所上市的公司数量 |
| **上市股票** | 上市交易的股票只数 | 只 | 比公司数多是因为有的公司发行了A股和B股等多种股票 |
| **流通市值** | 可交易股份的市场价值 | 亿元 | 只计算流通股的价值，比总市值更能反映真实可交易规模 |
| **报告时间** | 数据统计的截止日期 | YYYYMMDD | 数据更新的时间点 |
| **总股本** | 公司发行的全部股票数量 | 亿股 | 包括流通股和限售股（锁定不能卖的股份） |

---

## 市盈率（P/E）投资参考

| 市盈率范围 | 估值水平 | 投资含义 |
|-----------|---------|---------|
| < 10倍 | 低估 | 投资回报周期短，但可能意味着公司增长缓慢 |
| 10-20倍 | 合理 | 正常估值水平，投资回报周期适中 |
| 20-30倍 | 略高 | 市场对公司成长性有较高预期 |
| > 30倍 | 高估 | 高风险高预期，常见于成长股或热门板块 |

> **注意**：市盈率仅是参考指标，需结合行业、公司成长性、经济环境综合判断。

## 代码实现

### 获取数据
```python
import akshare as ak

# 获取上海证券交易所汇总数据
stock_sse_summary_df = ak.stock_sse_summary()
print(stock_sse_summary_df)

# 保存为CSV
stock_sse_summary_df.to_csv('sse_summary.csv', index=False, encoding='utf-8-sig')
```

### 读取与分析
```python
import pandas as pd

# 读取CSV
df = pd.read_csv('sse_summary.csv', encoding='utf-8-sig')

# 查看特定数据
total_market_cap = df[df['项目'] == '总市值']['股票'].values[0]
print(f"总市值: {total_market_cap}亿元")

# 计算流通比例
circulating = df[df['项目'] == '流通股本']['股票'].values[0]
total = df[df['项目'] == '总股本']['股票'].values[0]
ratio = (circulating / total) * 100
print(f"流通比例: {ratio:.2f}%")
```
