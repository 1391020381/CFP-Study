## 学习顺序建议

### Level I 推荐顺序
1. **Quant** → 打好数学基础
2. **FSA** → 财务报表分析（最重要）
3. **Equity** → 股票估值
4. **Fixed Income** → 债券基础
5. **Corporate** → 公司金融
6. **Portfolio** → 投资组合入门
7. **Derivatives** → 衍生品基础
8. **Alt** → 另类投资
9. **Economics** → 经济学
10. **Ethics** → 考前强化

### Level II 推荐顺序
1. **Equity Valuation** → 核心技能
2. **FSA** → 进阶财务分析
3. **Fixed Income** → 固定收益进阶
4. **Derivatives** → 衍生品估值
5. **Portfolio** → 投资组合
6. **Ethics** → 道德

### Level III 推荐顺序
1. **Portfolio Management** → 核心
2. **IPS** → 投资政策声明
3. **Behavioral** → 行为金融
4. **Institutions** → 机构投资者
5. **Ethics** → 道德
