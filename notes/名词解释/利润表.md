# 利润表 (Income Statement)

## 定义

**利润表** = 一张展示公司在**一段时间内**（如一年、一季度）赚了多少钱、花了多少钱、最后剩多少钱的"成绩单"。

简单来说：**公司的"业绩报告" —— 这段时间生意做得怎么样。**

---

## 核心逻辑

```
收入 - 成本 - 费用 = 利润
```

| 项目 | 含义 |
|------|------|
| 收入 | 卖货或提供服务收到的钱 |
| 成本 | 直接生产成本（原材料、人工） |
| 费用 | 间接费用（销售、管理、研发等） |
| 利润 | 最终剩下的钱 |

---

## 利润表的层次（从上到下）

### 第一层：营业收入

```
营业收入 = 主营业务收入 + 其他业务收入
```

**含义**：公司卖货或提供服务总共收到的钱。

---

### 第二层：毛利

```
毛利 = 营业收入 - 营业成本
毛利率 = 毛利 ÷ 营业收入
```

| 毛利率水平 | 含义 |
|-----------|------|
| > 50% | 很高（如茅台、奢侈品）✅ |
| 20%-50% | 正常（如制造业） |
| < 20% | 很低（如零售、竞争激烈行业）⚠️ |

---

### 第三层：营业利润

```
营业利润 = 毛利 - 销售费用 - 管理费用 - 研发费用 - 财务费用
```

| 费用类型 | 含义 | 例子 |
|---------|------|------|
| 销售费用 | 卖货花的钱 | 广告、销售人员工资 |
| 管理费用 | 管理公司花的钱 | 管理层工资、办公费 |
| 研发费用 | 研发投入 | 科技公司的重点 |
| 财务费用 | 借钱利息 | 利息支出 |

---

### 第四层：净利润

```
净利润 = 营业利润 + 营业外收入 - 营业外支出 - 所得税费用
```

**含义**：公司真正赚到的、归属于股东的钱。

---

## 一个直观的例子

假设茅台2023年简化版利润表：

| 项目 | 金额（亿元） |
|------|-------------|
| **营业收入** | **1505** |
| 减：营业成本 | 136 |
| **毛利** | **1369**（毛利率91%） |
| 减：税金及附加 | 144 |
| 减：销售费用 | 49 |
| 减：管理费用 | 103 |
| 减：研发费用 | 7 |
| **营业利润** | **1066** |
| 减：所得税 | 268 |
| **净利润** | **747** |

**解读**：
- 毛利率91% → 每卖100元酒，成本只有9元（超级高的定价权）✅
- 净利率50% → 每100元收入，净赚50元
- 茅台的利润质量：利润高、现金流也高

## 利润表的意义

### 1. 衡量盈利能力

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | 毛利 ÷ 营业收入 | 产品定价能力 |
| 净利率 | 净利润 ÷ 营业收入 | 最终赚钱能力 |
| ROE | 净利润 ÷ 股东权益 | 股东回报率 |

### 2. 判断成长性

```
收入增长率 = (今年收入 - 去年收入) ÷ 去年收入
净利润增长率 = (今年净利润 - 去年净利润) ÷ 去年净利润
```

| 增长率 | 评价 |
|--------|------|
| > 20% | 优秀 ✅ |
| 10%-20% | 良好 ✅ |
| 0%-10% | 一般 ⚠️ |
| < 0% | 下降 ❌ |

### 3. 发现问题信号

⚠️ **危险信号**：
- 收入增长但净利润下降（成本失控）
- 毛利率持续下降（竞争加剧或提价能力弱）
- 非经常性损益占比过高（主业不行，靠卖资产）

---

## 利润表的局限性

### ⚠️ 利润 ≠ 现金

| 场景 | 利润表 | 现金流量表 |
|------|--------|-----------|
| 赊销100万 | 收入+100万，利润+100万 | 0元（还没收到钱） |
| 预付租金 | 费用分期记 | 现金立即流出 |

**关键理解**：利润表基于**权责发生制**，交易发生就记账，不管钱是否真的收到。

### ⚠️ 容易被操纵

| 操纵手段 | 后果 |
|---------|------|
| 提前确认收入 | 虚增当期利润 |
| 延迟确认费用 | 虚增当期利润 |
| 变更折旧方法 | 调节利润 |

**记住**：现金流比利润更难造假！

---

## 关键财务指标（来自利润表）

### 1. 毛利率

```
毛利率 = 毛利 ÷ 营业收入 × 100%
```

| 毛利率 | 含义 | 典型行业 |
|--------|------|----------|
| > 50% | 定价权强 | 白酒、软件、奢侈品 |
| 20%-50% | 正常 | 制造业、医药 |
| < 20% | 竞争激烈 | 零售、建筑 |

### 2. 净利率

```
净利率 = 净利润 ÷ 营业收入 × 100%
```

| 净利率 | 含义 |
|--------|------|
| > 20% | 很优秀 ✅ |
| 10%-20% | 良好 |
| 5%-10% | 一般 |
| < 5% | 很低 ⚠️ |

### 3. 费用率

```
销售费用率 = 销售费用 ÷ 营业收入
管理费用率 = 管理费用 ÷ 营业收入
研发费用率 = 研发费用 ÷ 营业收入
```

## 与其他两张表的关系

```
┌─────────────────────────────────────────────────┐
│              三张表的勾稽关系                     │
├─────────────────────────────────────────────────┤
│                                                 │
│  利润表：本期净利润 100亿                        │
│      ↓                                         │
│  资产负债表：                                  │
│    - 现金 +100亿（如果分红了会更少）            │
│    - 留存收益 +100亿                            │
│      ↓                                         │
│  现金流量表：经营现金流 ≠ 净利润（有折旧等差异） │
│                                                 │
└─────────────────────────────────────────────────┘
```

**关键理解**：
- 利润表的净利润 → 增加资产负债表的留存收益
- 但现金不一定会增加同样的金额（因为可能有应收账款）

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握的概念：

1. **利润表的层次**：
   - 毛利
   - 营业利润
   - 净利润

2. **关键比率**：
   - 毛利率 = 毛利 ÷ 营业收入
   - 净利率 = 净利润 ÷ 营业收入
   - ROE = 净利润 ÷ 股东权益

3. **权责发生制**：
   - 收入在实现时确认
   - 费用在发生时确认
   - 不等于现金收支

4. **非经常性项目**：
   - 区分持续性和一次性项目
   - 评估"可持续利润"

5. **每股收益（EPS）**：
   ```
   EPS = (净利润 - 优先股股息) ÷ 加权平均股数
   ```

## 常见分析技巧

### 技巧1：看收入质量

| 好的收入 | 差的收入 |
|---------|---------|
| 有现金流支撑 | 都是应收账款 |
| 来自主营业务 | 来自卖资产 |
| 持续增长 | 忽高忽低 |

### 技巧2：看毛利率变化

```
毛利率 ↑ → 定价权增强 or 成本控制改善 ✅
毛利率 ↓ → 竞争加剧 or 成本上升 ⚠️
```

### 技巧3：看非经常性损益

```
扣非净利润 = 净利润 - 非经常性损益
```

如果净利润主要靠卖资产、政府补贴，说明主业不行。

## 实战案例：茅台 vs 普通公司

| 指标 | 茅台 | 普通制造业 |
|------|------|-----------|
| 毛利率 | 91% | 20% |
| 净利率 | 50% | 5% |
| 含金量 | 利润=现金 | 利润≠现金 |

**茅台为什么这么好？**
- 产品独特，定价权强
- 供不应求，经销商预付货款
- 成本固定，卖得越多赚得越多
