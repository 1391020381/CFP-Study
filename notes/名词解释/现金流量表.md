# 现金流量表 (Cash Flow Statement)

## 定义

**现金流量表** = 一张展示公司在**一段时间内**现金从哪里来、到哪里去的"流水账"。

简单来说：**公司的"钱包流水" —— 真正收到了多少现金，花了多少现金。**

---

## 核心逻辑

```
期末现金 = 期初现金 + 本期现金净增加额
```

**现金从哪来、到哪去？**
```
现金流入 - 现金流出 = 现金净增加额
```

---

## 现金流量表的三大活动

### 1. 经营活动现金流（CFO）

**含义**：公司**主业**产生的现金。

| 现金流入（正） | 现金流出（负） |
|---------------|---------------|
| 销售商品收到的现金 | 购买商品支付的现金 |
| 收到的税费返还 | 支付给职工的现金 |
| 收到其他与经营有关的现金 | 支付的各项税费 |
| | 支付其他与经营有关的现金 |

**判断标准**：
- CFO > 0 ✅ 主业在"造血"
- CFO < 0 ❌ 主业在"失血"

---

### 2. 投资活动现金流（CFI）

**含义**：公司**投资**（买设备、买公司）产生的现金。

| 现金流入（正） | 现金流出（负） |
|---------------|---------------|
| 收回投资收到的现金 | 购建固定资产支付的现金 |
| 取得投资收益收到的现金 | 投资支付的现金 |
| 处置固定资产收到的现金 | 取得子公司支付的现金 |

**判断标准**：
- 优质成长公司：CFI 通常是负的（持续投资）
- CFI > 0：可能在卖资产（不一定是好事）

---

### 3. 筹资活动现金流（CFF）

**含义**：公司**借钱或发股**产生的现金。

| 现金流入（正） | 现金流出（负） |
|---------------|---------------|
| 吸收投资收到的现金 | 偿还债务支付的现金 |
| 取得借款收到的现金 | 分配股利支付的现金 |
| 发行债券收到的现金 | |

**判断标准**：
- 初创公司：CFF > 0（在融资）
- 成熟公司：CFF < 0（在分红、还债）

---

## 一个直观的例子

假设茅台2023年简化版现金流量表：

| 项目 | 金额（亿元） |
|------|-------------|
| **经营活动现金流（CFO）** | **+765** |
| **投资活动现金流（CFI）** | **-55** |
| **筹资活动现金流（CFF）** | **-380**（分红） |
| **现金净增加额** | **+330** |

**解读**：
- CFO +765亿 → 主业"造血"能力超强 ✅
- CFI -55亿 → 正常投资扩产
- CFF -380亿 → 大额分红（回馈股东）✅
- 现金净增330亿 → 钱包更鼓了

## 现金流量表的意义

### 1. 判断公司"造血"能力

| CFO类型 | 含义 | 例子 |
|---------|------|------|
| CFO > 0 | 主业赚钱 | 茅台、苹果 |
| CFO ≈ 0 | 主业平衡 | 成熟稳定行业 |
| CFO < 0 | 主业失血 | 初创公司、亏损公司 |

### 2. 发现财务造假

**造假信号**：
- 利润表显示赚大钱，但CFO是负的 ⚠️
- 连续多年CFO < 净利润，应收账款激增 ⚠️
- CFI持续为正（卖资产维持现金流）⚠️

### 3. 评估公司生命周期

| 阶段 | CFO | CFI | CFF |
|------|-----|-----|-----|
| **初创期** | 负 | 负（投资）| 正（融资）|
| **成长期** | 正 | 负（扩产）| 正或负 |
| **成熟期** | 正 | 负或零 | 负（分红）|
| **衰退期** | 负或正 | 正（卖资产）| 负或零 |

## 关键财务指标（来自现金流量表）

### 1. 净利润现金比率

```
净利润现金比率 = 经营现金流 ÷ 净利润
```

| 数值 | 含义 |
|------|------|
| > 1 | 利润质量高 ✅ |
| ≈ 1 | 正常 |
| < 1 | 利润质量差 ⚠️ |
| < 0 | 危险 ❌ |

### 2. 自由现金流（FCF）

```
自由现金流 = 经营现金流 - 资本开支
```

**含义**：公司可以自由支配的现金（分红、回购、还债）

| FCF | 含义 |
|-----|------|
| > 0 | 有多余现金 ✅ |
| < 0 | 还需要投入 ⚠️ |

### 3. 现金分红比率

```
现金分红比率 = 分红金额 ÷ 经营现金流
```

| 数值 | 含义 |
|------|------|
| < 30% | 保守（留存发展）|
| 30%-50% | 适中 |
| > 50% | 慷慨（茅台这类）|

## 与利润表的区别（关键！）

| 对比 | 利润表 | 现金流量表 |
|------|--------|-----------|
| **原则** | 权责发生制 | 收付实现制 |
| **时间** | 交易发生时 | 真实收付时 |
| **能造假吗** | 相对容易 | 更难 |
| **反映什么** | 盈利能力 | 现金状况 |

### 举个例子：

| 场景 | 利润表 | 现金流量表 |
|------|--------|-----------|
| 卖货100万，客户赊账 | 收入+100万 | 0元 |
| 预付明年房租100万 | 费用分摊 | 现金流出100万 |
| 买入设备100万 | 折旧分期记 | 现金流出100万 |

## 与其他两张表的关系

```
┌─────────────────────────────────────────────────┐
│              三张表的勾稽关系                     │
├─────────────────────────────────────────────────┤
│                                                 │
│  利润表：净利润 100亿                            │
│      ↓                                         │
│  资产负债表：                                  │
│    - 现金期末余额 = 现金期初 + 现金净增加额     │
│      ↓                                         │
│  现金流量表：                                  │
│    - 现金净增加额 = CFO + CFI + CFF            │
│    - CFO ≈ 净利润 + 折旧 - 运营资本变动         │
│                                                 │
└─────────────────────────────────────────────────┘
```

**勾稽核对**：
```
现金流量表的期末现金 = 资产负债表的货币资金
```

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握的概念：

1. **三大活动**：
   - 经营活动（CFO）
   - 投资活动（CFI）
   - 筹资活动（CFF）

2. **直接法 vs 间接法**（编制CFO）：
   - 直接法：从现金收入/支出出发
   - 间接法：从净利润调整

3. **关键公式**：
   ```
   FCF = CFO - 资本开支
   净利润现金比率 = CFO ÷ 净利润
   ```

4. **现金流分析**：
   - CFO > 0 是好公司的标志
   - CFO > 净利润 是高质量利润

5. **三表勾稽关系**：
   - 现金流量表期末现金 = 资产负债表现金

## 常见分析技巧

### 技巧1：判断利润质量

```
高质量利润 = CFO > 净利润
低质量利润 = CFO < 净利润
```

### 技巧2：判断公司类型

| 公司类型 | CFO | CFI | CFF |
|---------|-----|-----|-----|
| 茅台类 | +++ | - | --（分红）|
| 成长类 | + | --- | + |
| 问题类 | - | + | + |

### 技巧3：发现财务造假信号

- 🚨 利润涨、CFO跌
- 🚨 连续多年CFO < 净利润
- 🚨 CFI持续为正（卖资产）
- 🚨 CFF持续为正（不断融资）

|------|------|---------|
| 净利润 | 747亿 | 100亿 |
| CFO | 765亿 | -50亿 |
| 净利润现金比率 | 1.02 | -0.5 |
| 评价 | 利润质量高 ✅ | 利润质量差 ❌ |

## 巴菲特的现金流观点

> **"现金流是企业的血液，利润只是会计数字。"**

**清崎的补充**：
> **"富人关注现金流，穷人关注'看起来很富有'。"**

### 实践建议：

1. **永远先看CFO**
   - CFO > 0 才是好公司

2. **CFO > 净利润**
   - 利润有真金白银支撑

3. **警惕CFO持续为负**
   - 即使利润很高也可能是纸面富贵
