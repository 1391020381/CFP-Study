# GDP（国内生产总值）

## 定义

**GDP（Gross Domestic Product，国内生产总值）**是指一个国家/地区在一定时期内生产的**所有最终产品和服务的市场价值**。

简单来说：**一个国家一年创造了多少价值**

## 核心要点

| 要点 | 说明 |
|------|------|
| **地域性** | 境内生产，不管企业是本国还是外国 |
| **最终产品** | 只计算最终产品，不计算中间品（避免重复） |
| **市场价值** | 按市场价格计算 |
| **一定时期** | 通常是一年或一个季度 |

## GDP 的计算方法

### 方法1：支出法（最常用）
```
GDP = C + I + G + (X - M)
```

| 组成 | 含义 | 例子 |
|------|------|------|
| **C** | 消费 | 居民消费支出 |
| **I** | 投资 | 企业投资、房地产 |
| **G** | 政府支出 | 政府购买、基础设施 |
| **X** | 出口 | 出口商品和服务 |
| **M** | 进口 | 进口商品和服务 |

### 方法2：收入法
```
GDP = 工资 + 利润 + 利息 + 租金 + 折旧 + 税收
```

### 方法3：生产法
按行业增加值计算

## GDP 的含义

### GDP 增长率

| GDP 变化 | 含义 |
|----------|------|
| **GDP 增长（正）** | 经济扩张 |
| **GDP 下降（负）** | 经济收缩 |
| **GDP 持续2个季度下降** | 技术性衰退 |

### GDP 增速分类

| 增速 | 评价 |
|------|------|
| < 3% | 增长缓慢 |
| 3-5% | 健康增长 |
| 5-8% | 快速增长 |
| > 8% | 过热风险 |
| < 0% | 衰退 |

## 实际 GDP vs 名义 GDP

| 类型 | 定义 | 用途 |
|------|------|------|
| **名义GDP** | 按当前价格计算 | 反映经济总量 |
| **实际GDP** | 剔除通胀因素 | 反映真实增长 |

```
实际GDP = 名义GDP / GDP平减指数
```

**实际GDP增长率才是真实的经济增长**

## 中国 GDP

### 政府目标
- 政府工作报告设定 GDP 增长目标（如"5%左右"）
- 目标具有指导意义

### 中国 GDP 特点
1. **投资驱动明显**
   - 固定资产投资占比较高

2. **消费占比上升**
   - 消费对GDP贡献逐年增加

3. **第三产业主导**
   - 服务业占比超过50%

### 各国 GDP 比较（2023年参考）

|------|------|-----------------|
| 1 | 美国 | 约27 |
| 2 | 中国 | 约18 |
| 3 | 德国 | 约4.4 |

## GDP 与投资

### GDP 增长对股市的影响

| GDP 状况 | 股市影响 | 原因 |
|----------|----------|------|
| 快速增长 | 🟢 利好 | 企业盈利改善 |
| 增长放缓 | 🟡 谨慎 | 盈利压力 |
| 负增长 | 🔴 利空 | 企业盈利下滑 |

### GDP 与利率

| GDP 状况 | 利率政策 | 原因 |
|----------|----------|------|
| 过热 | 加息 | 防止通胀 |
| 衰退 | 降息 | 刺激经济 |

## GDP 的局限

| 局限 | 说明 |
|------|------|
| **不反映收入分配** | GDP增长不代表人人受益 |
| **不反映福利** | 不考虑环境、闲暇时间 |
| **地下经济** | 不计入非正式经济活动 |
| **质量变化** | 难以衡量产品质量提升 |

## 其他相关指标

| 指标 | 全称 | 说明 |
|------|------|------|
| **GNP** | 国民生产总值 | 本国公民生产，不管在境内还是境外 |
| **PPP** | 购买力平价 | 考虑物价水平的国际比较 |
| **人均GDP** | GDP/人口 | 反映平均生活水平 |

## CFP 考试应用

| 应用场景 | 说明 |
|----------|------|
| **判断经济状况** | GDP增速反映经济健康度 |
| **预测企业盈利** | GDP增长 → 企业盈利改善 |
| **评估投资时机** | GDP扩张期通常利好股票 |
| **利率政策预测** | GDP过热 → 可能加息 |

## 记忆技巧

```
GDP = 一个国家一年生产多少值钱的东西

支出法：C + I + G + (X - M)
C = 消费（老百姓花钱）
I = 投资（企业花钱）
G = 政府（政府花钱）
X - M = 净出口（卖出去 - 买进来）

GDP ↑ → 经济好 → 股市通常好
GDP ↓ → 经济差 → 股市通常差
```

## 关键要点

✅ **GDP 衡量一个国家的经济总量**
✅ **实际GDP增长率才是真实增长**
✅ **GDP 持续2个季度下降 = 衰退**
✅ **GDP 与股市、利率政策密切相关**
