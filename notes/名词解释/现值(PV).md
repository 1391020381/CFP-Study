# 现值 (PV)

## 定义

**现值（Present Value, PV）**是指未来的一笔钱**折算到今天**值多少。

## 核心概念

- **"倒推"**：从未来推算到现在
- 考虑时间价值和投资回报率
- 用于评估未来现金流在今天的价值

## 计算公式

```
PV = FV / (1 + r)^n
```

**其中**：
- **PV** = 现值
- **FV** = 终值（未来金额）
- **r** = 每期利率（折现率）
- **n** = 期数

## 实际例子

### 例1：退休规划
> **问题**：你10年后需要50万，年化收益6%，现在需要存多少？
>
> **计算**：PV = 50 / (1.06)^10 ≈ 27.92万
>
> **答案**：现在需要约27.92万

### 例2：房贷评估
> **问题**：未来30年每月还1万房贷，按6%折现，这笔房贷今天值多少钱？
>
> 这是**年金现值**问题，需要用年金公式

## CFP 考试应用

| 应用场景 | 说明 |
|----------|------|
| **退休规划** | 计算退休后需要的养老金现值 |
| **房贷评估** | 计算未来还款总额的现值 |
| **保险需求分析** | 计算未来保障需求的现值 |
| **投资评估** | 判断投资是否值得（现值 > 成本） |

## 与终值的关系

```
现值 --(复利增长)--> 终值
终值 --(折现)-->     现值
```

## 常见错误

❌ **错误1**：认为100万未来值和100万现在值一样
- 正确：需要折现计算

❌ **错误2**：折现率选择错误
- 正确：应该用合适的回报率作为折现率

❌ **错误3**：忽略期数
- 正确：注意是年、月还是季度

## 关键要点

✅ **现值是"倒推"计算**
✅ **折现率越高，现值越低**
✅ **时间越长，现值越低**
✅ **CFP考试中必考概念**
