# 每股收益 (Earnings Per Share, EPS)

## 定义

**每股收益（EPS）** = 一家公司在一定时期内（通常是一年或一季度）**每一股股票能分到多少净利润**。

简单来说：**如果你持有这家公司的一股，这股股票"赚"了多少钱。**

---

## 计算公式

```
EPS = 净利润 ÷ 总股本
```

| 指标 | 含义 |
|------|------|
| 净利润 | 公司扣除所有成本、费用、税收后，真正赚到的钱 |
| 总股本 | 公司发行的所有股票数量 |

---

## 一个直观的例子

假设有一家公司 A：

| 项目 | 数值 |
|------|------|
| 公司一年净利润 | 10亿元 |
| 公司发行的总股数 | 10亿股 |

```
EPS = 10亿 ÷ 10亿 = 1元/股
```

**含义**：每一股股票"赚"了1块钱。

---

## EPS 的意义

### 1. 衡量公司盈利能力

EPS 越高，说明每一股股票"赚"的钱越多，公司的盈利能力越强。

| 公司 | EPS | 评价 |
|------|-----|------|
| A公司 | 5元/股 | ✅ 盈利能力强 |
| B公司 | 0.5元/股 | ⚠️ 盈利能力一般 |
| C公司 | -1元/股 | ❌ 亏损（EPS为负） |

### 2. 判断公司成长性

看 EPS 的**变化趋势**比看单期数值更重要：

| 情况 | EPS变化 | 含义 |
|------|---------|------|
| 持续增长 | 1元 → 1.5元 → 2元 | ✅ 公司越来越好 |
| 持续下降 | 2元 → 1.5元 → 1元 | ❌ 公司在走下坡路 |
| 忽高忽低 | 2元 → 0.5元 → 3元 | ⚠️ 业绩不稳定 |

### 3. 计算市盈率（PE）的基础

```
市盈率（PE）= 股价 ÷ EPS
```

**EPS 是计算 PE 的基础数据**，没有 EPS 就无法判断股价是否合理。

---

## EPS 的局限性

### ⚠️ EPS ≠ 分红

**重要误解**：很多人认为 EPS 就是公司会分给股东的钱。

| 概念 | 含义 |
|------|------|
| EPS | 每一股"赚"的钱（可能留存用于发展） |
| 分红（股息） | 每一股真正分到手里的现金 |

**例子**：
- EPS = 2元
- 但公司可能只分红 0.5元，剩下的1.5元用于再投资

### ⚠️ EPS 可以被"美化"

公司可以通过会计手段让 EPS 看起来更好，但实际现金流可能很差。

**记住清崎的教导**：关注现金流，而不只是账面利润！

### ⚠️ EPS 不能跨行业比较

不同行业的 EPS 没有可比性：

| 行业 | 正常EPS水平 | 原因 |
|------|-------------|------|
| 银行、房地产 | 较高（2-5元） | 资产密集型 |
| 科技、互联网 | 较低（0.5-2元） | 成长性，注重未来 |
| 零售、制造业 | 中等（1-3元） | 稳定型 |

## 如何使用 EPS 分析股票

### 1. 看历史趋势

```
某公司近5年EPS：
2020年：1.2元
2021年：1.5元  ✅ 增长25%
2022年：1.8元  ✅ 增长20%
2023年：2.3元  ✅ 增长28%
2024年：2.9元  ✅ 增长26%
```

**判断**：公司盈利能力持续增强，是好迹象。

### 2. 看 EPS 增长率

```
EPS增长率 = (今年EPS - 去年EPS) ÷ 去年EPS × 100%
```

| 增长率 | 评价 |
|--------|------|
| >20% | 优秀 ✅ |
| 10%-20% | 良好 ✅ |
| 0%-10% | 一般 ⚠️ |
| <0% | 下降 ❌ |

### 3. 结合市盈率（PE）判断估值

| 情况 | PE | EPS增长 | 判断 |
|------|-----|---------|------|
| 低 | 10倍 | 15% | ✅ 被低估，可以考虑 |
| 高 | 50倍 | 50% | ⚠️ 可能合理（高成长） |
| 高 | 50倍 | 5% | ❌ 可能高估（泡沫） |

## 与《富爸爸穷爸爸》理念的结合

### 清崎的观点：

> **"富人关注现金流，穷人关注'看起来很富有'。"**

### 如何用清崎思维看 EPS：

| 传统视角 | 清崎视角 |
|----------|----------|
| 只看 EPS 高低 | 看 EPS 的"质量"（是否有真实现金流支撑） |
| 只看每股赚多少 | 看公司是否把利润转化为再投资能力 |
| 只看过去 EPS | 看 EPS 增长的可持续性 |

### 实践建议：

1. **EPS + 经营现金流** 一起看
   - EPS 高 + 经营现金流高 = 优质公司 ✅
   - EPS 高 + 经营现金流低 = 可能有问题 ⚠️

2. **EPS + 股息** 一起看
   - EPS 高 + 股息稳定增长 = 股东友好型公司 ✅
   - EPS 高 + 不分红 = 需要看资金用途 ⚠️

3. **EPS + 行业地位** 一起看
   - 龙头公司的 EPS 更有持续性 ✅
   - 小公司的 EPS 波动性大 ⚠️
