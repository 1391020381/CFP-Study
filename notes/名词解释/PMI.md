# PMI（采购经理人指数）

## 定义

**PMI（Purchasing Managers' Index，采购经理人指数）**是衡量制造业/服务业景气程度的**先行指标**。

简单来说：**调查企业采购经理，了解他们所在行业的景气状况**

## PMI 的核心含义

| PMI 值 | 含义 | 经济状况 |
|--------|------|----------|
| **PMI > 50** | 行业在**扩张** | 经济向好 |
| **PMI = 50** | 无变化 | 临界点 |
| **PMI < 50** | 行业在**收缩** | 经济向差 |

### PMI 与荣枯线

```
PMI = 50 → 荣枯线
> 50 → 荣（扩张）
< 50 → 枯（收缩）
```

## PMI 的类型

| 类型 | 全称 | 说明 |
|------|------|------|
| **制造业PMI** | 制造业采购经理指数 | 衡量工厂生产活动 |
| **非制造业PMI** | 服务业采购经理指数 | 衡量服务业活动 |

## PMI 的构成

PMI 由5个分项指数加权计算：

| 分项 | 权重 | 说明 |
|------|------|------|
| **新订单** | 30% | 反映需求 |
| **生产** | 25% | 反映生产活动 |
| **就业** | 20% | 反映就业状况 |
| **供应商配送** | 15% | 反映供应链效率 |
| **库存** | 10% | 反映库存水平 |

## PMI 的特点

### 1. 先行指标
- **比 GDP 更早**反映经济变化
- 领先 GDP 约 1-3 个月

### 2. 发布及时
- **每月发布**（次月初）
- 比季度 GDP 更及时

### 3. 动态变化
- 连续上升 → 经济改善
- 连续下降 → 经济恶化

## 实际例子

### 例1：经济扩张
> 2024年1月制造业PMI = 50.8%
>
> **解读**：
> - 50.8% > 50%，制造业在扩张
> - 但刚超过50，扩张力度不大
> - 经济有复苏迹象

### 例2：经济收缩
> 2024年1月制造业PMI = 49.2%
>
> **解读**：
> - 49.2% < 50%，制造业在收缩
> - 经济面临下行压力
> - 央行可能考虑宽松政策

### 例3：强劲扩张
> 制造业PMI从48% → 50% → 52%
>
> **解读**：
> - 连续3个月上升
> - 从收缩进入扩张
> - 经济明显好转

## PMI 与 GDP 的关系

```
PMI > 50 → GDP 可能增长
PMI < 50 → GDP 可能下降
```

| PMI | GDP |
|-----|-----|
| 连续 > 50 | 通常正增长 |
| 连续 < 50 | 可能负增长 |

## PMI 与股市

| PMI 变化 | 股市影响 | 原因 |
|----------|----------|------|
| PMI 超过50 | 🟢 利好 | 经济扩张，企业盈利改善 |
| PMI 低于50 | 🔴 利空 | 经济收缩，企业盈利下滑 |
| PMI 大幅上升 | 🟢 利好 | 经济加速复苏 |

## CFP 考试应用

| 应用场景 | 说明 |
|----------|------|
| **预测经济走势** | PMI > 50 → GDP 可能增长 |
| **判断行业景气** | 制造业PMI vs 非制造业PMI |
| **投资时机** | PMI上升期 → 增加股票配置 |
| **利率政策预测** | PMI持续低位 → 可能降息 |

## 中国 PMI 特点

1. **官方 PMI vs 财新 PMI**
   - 官方PMI：调查大型企业
   - 财新PMI：调查中小型企业

2. **两 PMI 分化**
   - 官方PMI > 财新PMI：大型企业好于中小企业
   - 反之：中小企业更有活力

## PMI 的局限

| 局限 | 说明 |
|------|------|
| **样本限制** | 只反映采购经理的看法 |
| **行业覆盖** | 服务业PMI可能不够准确 |
| **短期波动** | 需要连续观察，单月可能失真 |

## 综合分析

### 多指标结合
1. **PMI 趋势**：连续3个月以上才有意义
2. **与 GDP 对比**：验证 PMI 的预测准确性
3. **与其他指标结合**：结合 CPI、PPI、就业数据

## 记忆技巧

```
PMI = 采购经理信心指数

50 = 荣枯线
> 50 = 荣（扩张，有活力）
< 50 = 枯（收缩，不景气）

PMI 先行，GDP 滞后
PMI 说了 → GDP 后来才表现出来

PMI ↑ → 经济好 → 股市通常好
PMI ↓ → 经济差 → 股市通常差
```

## 常见错误

❌ **错误1**：单月PMI下结论
- 正确：需要连续观察趋势

❌ **错误2**：忽视制造业与非制造业差异
- 正确：两者要分开分析

❌ **错误3**：PMI略高于50就过度乐观
- 正确：要看PMI的数值和趋势

## 关键要点

✅ **PMI = 50 是荣枯线**
✅ **PMI > 50 扩张，< 50 收缩**
✅ **PMI 是先行指标，领先 GDP**
✅ **需要连续观察，不要单月下结论**
