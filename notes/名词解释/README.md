## 学习建议

1. **先掌握 TVM 基础**
   - 货币时间价值是所有计算的基础
   - 现值、终值、年金必须熟练

2. **理解经济指标的含义**
   - CPI、PMI、PPI、GDP 的关系
   - 如何解读这些数据

3. **结合实际应用**
   - 用自己的120w资产做练习
   - 思考这些概念如何影响投资决策

4. **不要死记公式**
   - CFP考试会提供公式表
   - 重要的是理解何时用哪个公式

## 名词关系图

```
货币时间价值 (TVM)
├─ 现值 (PV) ────────┐
├─ 终值 (FV) ────────┤
├─ 年金 ──────────────┤─ 相互转换计算
└─ 复利 ──────────────┘

经济指标体系
├─ 通胀衡量
│  ├─ CPI (消费品价格)
│  └─ PPI (工业品价格) → 领先 CPI
├─ 经济总量
│  └─ GDP
├─ 先行指标
│  └─ PMI → 领先 GDP
└─ 通胀调整
   └─ 名义利率 → 实际利率

经济周期
├─ 复苏 → PMI > 50, GDP ↑
├─ 扩张 → CPI ↑, 可能加息
├─ 衰退 → PMI < 50, GDP ↓
└─ 萧条 → 可能降息

投资决策
├─ 机会成本 → 比较不同选择
├─ 实际利率 → 真实收益
└─ 经济周期 → 资产配置
```
