# 消费者物价指数 (CPI)

## 定义

**CPI（Consumer Price Index，消费者物价指数）**是衡量**一篮子消费品和服务价格变化**的指标，是**通胀率的主要衡量标准**。

## CPI 是什么？

### "一篮子"商品
CPI 统计一篮子常见商品和服务的价格变化，包括：

| 类别 | 内容 |
|------|------|
| **食品** | 米面油、肉蛋奶、蔬菜水果 |
| **居住** | 房租、水电燃料 |
| **交通** | 汽油、公共交通、汽车 |
| **医疗** | 医药费用、医疗服务 |
| **教育** | 学费、书本费 |
| **衣着** | 服装、鞋类 |
| **其他** | 通讯、娱乐等 |

### 统计方式
1. **基期设定**：某年=100（基准）
2. **定期调查**：每月收集价格数据
3. **计算指数**：当前价格 vs 基期价格

## CPI 的含义

| CPI 变化 | 含义 | 通胀率 |
|----------|------|--------|
| CPI 从100涨到102 | 物价上涨2% | 2% |
| CPI 从100涨到103 | 物价上涨3% | 3% |
| CPI 从100跌到99 | 物价下跌1% | -1%（通缩） |

**CPI上涨幅度 = 通胀率**

## 实际例子

### 例1：购买力变化
> 2023年CPI = 100，2024年CPI = 103
>
> **解读**：
> - 通胀率 = 3%
> - 2024年100元 = 2023年的97元（100/103）
> - 购买力下降约3%

### 例2：工资实际增长
> 工资上涨5%，CPI上涨3%
>
> **解读**：
> - 名义工资增长：5%
> - **实际工资增长**：5% - 3% = **2%**
> - 购买力实际增加2%

## CPI 与投资

### 实际收益计算
```
实际收益 = 名义收益 - CPI涨幅
```

| 投资收益 | CPI | 实际收益 | 评价 |
|----------|-----|----------|------|
| 4% | 2% | 2% | ✅ 正收益 |
| 4% | 3% | 1% | ✅ 微弱正收益 |
| 4% | 4% | 0% | ⚠️ 刚保本 |
| 4% | 5% | -1% | ❌ 购买力下降 |

## CPI 的局限

| 局限 | 说明 |
|------|------|
| **平均化** | 不能反映个人实际消费结构 |
| **滞后性** | 数据发布有延迟 |
| **不包含资产** | 房价、股价不计入CPI |
| **地区差异** | 全国平均，不代表地方 |

## CPI vs PPI

| 指标 | 全称 | 衡量对象 | 影响 |
|------|------|----------|------|
| **CPI** | 消费者物价指数 | 消费品价格 | 直接影响生活 |
| **PPI** | 生产者物价指数 | 工业品出厂价 | 预测未来CPI |

**传导关系**：
```
PPI上涨 → 企业成本上升 → 产品价格上升 → CPI上涨
```

## CFP 考试应用

| 应用场景 | 说明 |
|----------|------|
| **判断通胀** | CPI涨幅 = 通胀率 |
| **实际回报** | 收益率 - CPI = 实际收益 |
| **调整规划** | 长期财务目标考虑CPI |
| **经济分析** | CPI过高 → 可能加息 |

## CPI 的合理水平

| CPI涨幅 | 评价 | 影响 |
|---------|------|------|
| 0-1% | 偏低 | 可能有通缩压力 |
| 1-2% | 健康 | 经济稳定 |
| 2-3% | 温和 | 可接受 |
| 3-5% | 偏高 | 需要关注 |
| >5% | 严重 | 需要控制 |

## 中国 CPI 特点

1. **食品权重高**：食品占比较大，猪肉价格影响明显
2. **不含房价**：房租计入，但房价上涨不计入
3. **目标控制**：政府通常设定"3%左右"的调控目标

## 常见错误

❌ **错误1**：认为CPI反映个人实际感受
- 正确：CPI是平均值，个人感受可能不同

❌ **错误2**：忽视CPI对长期规划的影响
- 正确：10年以上必须考虑CPI

❌ **错误3**：混淆CPI和房价
- 正确：房价不计入CPI，但房租计入

## 关键要点

✅ **CPI = 通胀率**
✅ **实际收益 = 名义收益 - CPI**
✅ **长期规划必须考虑CPI**
✅ **CPI是滞后指标**
