# 货币时间价值 (TVM)

## 定义

**货币时间价值（Time Value of Money, TVM）**是指**今天的一笔钱比未来同等金额的钱更有价值**这一金融概念。

## 核心原理

| 原因 | 解释 |
|------|------|
| **通货膨胀** | 物价上涨，货币购买力下降 |
| **机会成本** | 今天的钱可以投资赚钱，未来才能收到的钱失去了这个机会 |

## 为什么今天的钱更值钱？

### 1. 通货膨胀
- 今天100元能买10斤米
- 一年后通胀3%，100元只能买9.7斤米
- 购买力下降

### 2. 机会成本
- 今天100元存银行，一年后变成103元
- 一年后才给你100元，你失去了赚3元的机会
- 所以今天的100元 > 一年后的100元

## CFP 考试应用

| 应用场景 | 说明 |
|----------|------|
| **退休规划** | 计算未来需要多少养老金 |
| **教育规划** | 计算现在需要存多少才能满足未来教育费用 |
| **房贷计算** | 计算每月还款额的现值 |
| **投资决策** | 比较不同投资方案的真实价值 |

## 相关概念

- **现值 (PV)**：未来的钱折算到现在
- **终值 (FV)**：现在的钱计算到未来
- **折现率**：用于计算现值的利率
- **复利**：TVM的计算基础

## 经典名言

> "时间就是金钱" — 本杰明·富兰克林

## 关键要点

✅ **TVM 是金融数学的基础**
✅ **所有投资决策都应考虑时间价值**
✅ **跨时间比较必须折算到同一时点**
