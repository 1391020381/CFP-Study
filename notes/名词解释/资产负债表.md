# 资产负债表 (Balance Sheet)

## 定义

**资产负债表** = 一张展示公司在**某个特定时点**拥有什么（资产）、欠别人什么（负债）、股东真正拥有什么（所有者权益）的快照。

简单来说：**公司的"家底"清单 —— 钱从哪来，用到哪去了。**

---

## 核心公式

```
资产 = 负债 + 所有者权益
```

| 项目 | 含义 |
|------|------|
| 资产 | 公司拥有的、能带来未来经济利益的资源 |
| 负债 | 公司欠别人的、需要偿还的债务 |
| 所有者权益 | 股东真正拥有的公司价值 |

**换个角度理解**：
```
钱怎么用的（资产） = 钱从哪来的（负债 + 股东投入）
```

---

## 资产负债表的结构

### 左边：资产（钱的用途）

| 资产类型 | 含义 | 例子 |
|---------|------|------|
| **流动资产** | 一年内能变现的资产 | 现金、应收账款、存货 |
| **非流动资产** | 持有一年以上的资产 | 固定资产、无形资产、长期投资 |

### 右边：负债（钱从哪来 - 借的）

| 负债类型 | 含义 | 例子 |
|---------|------|------|
| **流动负债** | 一年内要还的债 | 应付账款、短期借款 |
| **非流动负债** | 一年后才还的债 | 长期借款、债券 |

### 右边：所有者权益（钱从哪来 - 股东的）

| 权益项目 | 含义 |
|---------|------|
| **股本** | 股东投入的本金 |
| **留存收益** | 公司历年赚了但没分掉的钱 |

---

## 一个直观的例子

假设茅台2023年简化版资产负债表：

| 项目 | 金额（亿元） |
|------|-------------|
| **资产** | **2543** |
| - 现金 | 690 |
| - 应收账款 | 1 |
| - 存货 | 418 |
| - 固定资产 | 200 |
| **负债** | **453** |
| - 应付账款 | 120 |
| - 预收账款 | 150 |
| **所有者权益** | **2090** |
| - 股本 | 13 |
| - 留存收益 | 2000+ |

**验证公式**：2543（资产）= 453（负债）+ 2090（权益）✅

**解读**：
- 茅台每1元资产中，只有0.18元是借的（负债率低）
- 股东权益占82%（公司真正属于股东的）
- 现金占资产27%（现金流很好）

## 资产负债表的意义

### 1. 判断公司财务健康度

| 指标 | 好公司 | 差公司 |
|------|--------|--------|
| 资产负债率 | < 50% | > 70% |
| 流动比率 | > 2 | < 1 |
| 现金占比 | > 10% | 很低 |

### 2. 发现公司经营特点

| 公司类型 | 资产负债表特征 |
|---------|----------------|
| 茅台类 | 现金多、应收账款少、预收账款多 |
| 零售类 | 存货多、固定资产多 |
| 互联网类 | 现金多、无形资产多 |
| 制造业类 | 固定资产多、应收账款多 |

### 3. 识别风险信号

⚠️ **危险信号**：
- 应收账款占比过高（卖货收不到钱）
- 存货积压（货卖不出去）
- 短期债务激增（资金链紧张）
- 商誉过高（可能减值）

---

## 关键财务指标（来自资产负债表）

### 1. 资产负债率

```
资产负债率 = 总负债 ÷ 总资产
```

| 数值 | 含义 |
|------|------|
| < 30% | 很保守 |
| 30%-50% | 健康区间 |
| 50%-70% | 注意风险 |
| > 70% | 风险较高 |

### 2. 流动比率

```
流动比率 = 流动资产 ÷ 流动负债
```

| 数值 | 含义 |
|------|------|
| > 2 | 偿债能力强 ✅ |
| 1-2 | 正常 |
| < 1 | 偿债压力大 ⚠️ |

---

## 资产负债表的局限性

### ⚠️ 历史成本 vs 真实价值

| 项目 | 资产负债表显示 | 真实情况 |
|------|---------------|----------|
| 茅台的老厂房 | 10年前建造成本 | 现在可能值100倍 |
| 土地使用权 | 购买成本 | 现在可能升值很多 |

### ⚠️ 不包括"隐形资产"

- 品牌价值
- 客户关系
- 技术团队
- 市场地位

## 与其他两张表的关系

```
┌─────────────────────────────────────────────────┐
│              三张表的勾稽关系                     │
├─────────────────────────────────────────────────┤
│                                                 │
│  利润表：本年净利润 100亿                        │
│      ↓                                         │
│  资产负债表：现金 +100亿 或 留存收益 +100亿      │
│      ↓                                         │
│  现金流量表：经营现金流 80亿（可能有折旧等差异）  │
│                                                 │
└─────────────────────────────────────────────────┘
```

**关键理解**：
- 利润表是"视频"（一段时间）
- 资产负债表是"照片"（某个时点）
- 净利润增加 → 留存收益增加 → 所有者权益增加

## CFA 考试重点

### 考试频率：⭐⭐⭐⭐⭐

### 必须掌握的概念：

1. **会计恒等式**：资产 = 负债 + 所有者权益

2. **资产分类**：
   - 流动资产 vs 非流动资产
   - 各自包含的科目

3. **负债分类**：
   - 流动负债 vs 非流动负债
   - 各自包含的科目

4. **所有者权益构成**：
   - 股本
   - 资本公积
   - 留存收益

5. **关键比率**：
   - 资产负债率 = 总负债 ÷ 总资产
   - 流动比率 = 流动资产 ÷ 流动负债
   - 速动比率 = (流动资产 - 存货) ÷ 流动负债

## 常见分析技巧

### 技巧1：应收账款分析

```
应收账款周转天数 = 365 ÷ (收入 ÷ 平均应收账款)
```

| 天数 | 含义 |
|------|------|
| < 30天 | 很好（回款快）✅ |
| 30-60天 | 正常 |
| > 90天 | 要注意 ⚠️ |

### 技巧2：存货分析

```
存货周转天数 = 365 ÷ (营业成本 ÷ 平均存货)
```

| 天数 | 含义 |
|------|------|
| < 60天 | 周转快 ✅ |
| 60-120天 | 正常 |
| > 180天 | 积压风险 ⚠️ |

### 技巧3：看钱的来源

```
(年末现金 - 年初现金) = 经营现金流 + 投资现金流 + 筹资现金流
```
