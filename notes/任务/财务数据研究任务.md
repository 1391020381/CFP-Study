# 财务数据研究任务

## 任务目标

深入研究三家公司的财务数据，应用《富爸爸穷爸爸》第二课学到的财务知识，验证投资选择的合理性。

---

## 研究对象

### 1. 贵州茅台（600519）
**选择理由**：
- 业务模式好，成本低，利润高
- 用户有粘性强，品牌护城河深

**研究重点**：
- 验证"高利润"的可持续性
- EPS是否稳定增长？
- 经营现金流是否匹配净利润？

### 2. 宁德时代（300750）
**选择理由**：
- 电池技术领先
- 市占比高，规模效应明显
- 行业龙头，业绩优秀

**研究重点**：
- 评估成长性
- EPS增长率是否超过20%？
- 行业景气度如何影响业绩？

### 3. 比亚迪（002594）
**选择理由**：
- 新能源汽车技术领先
- 产业链可控，垂直整合
- 业务高速增长

**研究重点**：
- 判断增长质量
- 高增长是否带来真实现金流？
- 还是只是"账面增长"？

## 清崎财务思维检查清单

在分析每家公司时，问自己这些问题：

### ✅ 现金流视角
- [ ] 这家公司赚钱吗？（EPS > 0）
- [ ] 赚的钱收到现金了吗？（经营现金流 > 净利润）
- [ ] 现金流稳定吗？（看历史趋势）

### ✅ 资产质量
- [ ] 公司的资产能产生收入吗？（ROE > 15%）
- [ ] 负债是否过高？（资产负债率 < 60%为佳）
- [ ] 公司在持续投入资产吗？（资本开支情况）

### ✅ 盈利质量
- [ ] 利润来自主营业务还是投资收益？
- [ ] 利润可持续吗？（行业地位、护城河）
- [ ] 行业前景如何？（未来3-5年）
