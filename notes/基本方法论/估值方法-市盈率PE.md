# 估值方法-市盈率（PE）详解

## 一、基本定义

### 市盈率（Price-to-Earnings Ratio）

```
市盈率 PE = 股价 ÷ 每股收益
         = P ÷ E

含义：投资回本需要多少年
```

**示例：**
```
股价 = 50元
每股收益 = 5元
PE = 50 ÷ 5 = 10倍

意思：按当前盈利水平，
      10年能赚回投资本金
```

---

## 二、市盈率的三种类型

### 1. 静态市盈率（LYR，Last Year Ratio）

```
静态PE = 股价 ÷ 上一年度EPS

示例：
当前股价：38.67元
2023年EPS：5.66元
静态PE = 38.67 ÷ 5.66 = 6.83倍
```

**特点：**
- 使用年报数据，数据确定
- 更新频率低（一年一次）
- 有滞后性（可能是去年的业绩）

**适用场景：**
- 业绩稳定的公司
- 年度对比分析
- 验证数据准确性

---

### 2. 动态市盈率 / TTM市盈率

```
TTM PE = 股价 ÷ TTM EPS

TTM EPS = 最近4个季度EPS之和
        = Q4去年 + Q1今年 + Q2今年 + Q3今年
```

**示例（招商银行2024年Q3）：**
```
2023年Q4单季EPS：1.41元
2024年Q1单季EPS：1.52元
2024年Q2单季EPS：1.46元
2024年Q3单季EPS：1.53元

TTM EPS = 1.41 + 1.52 + 1.46 + 1.53 = 5.92元

TTM PE = 38.67 ÷ 5.92 = 6.53倍
```

**为什么叫TTM？**
```
TTM = Trailing Twelve Months
     = 过去十二个月
     = 滚动年度
     = 移动年度
```

**特点：**
- 使用最近4个季度数据
- 更新频率高（每季更新）
- 更贴近当前经营状况
- 消除季节性因素
- **主流平台使用（同花顺、雪球等）** ✅

**为什么TTM最好？**
```
静态PE：数据旧，可能过时
TTM PE：数据新，反映现状
预期PE：数据虚，可能不准

→ TTM PE平衡了及时性和准确性
```

---

### 3. 预期市盈率（Forward PE）

```
预期PE = 股价 ÷ 预期未来EPS

示例：
当前股价：38.67元
2025年预期EPS：6.20元
预期PE = 38.67 ÷ 6.20 = 6.24倍
```

**特点：**
- 使用预测数据
- 反映未来估值
- 不确定性高
- 依赖分析师预测

**来源：**
- 分析师一致预期
- 公司业绩指引
- 投资者自行预测

---

## 三、三种PE对比

| 类型 | EPS来源 | 更新频率 | 优点 | 缺点 | 使用场景 |
|------|---------|----------|------|------|----------|
| **静态PE** | 去年年报 | 一年 | 数据确定 | 滞后明显 | 业绩稳定公司 |
| **TTM PE** | 最近4季度 | 每季 | 及时准确 | 计算复杂 | **最常用** ✅ |
| **预期PE** | 未来预测 | 随时 | 面向未来 | 不确定 | 成长股分析 |

## 四、统计术语解释

### 1. 分位值（Percentile）

**定义：** 当前估值在历史中所处的位置

```
分位值 = 小于当前值的天数 ÷ 总天数 × 100%
```

**示例（雪球APP显示）：**
```
招商银行PE分位值：15%

含义：
- 历史上只有15%的时间PE低于当前
- 当前处于历史低位
- 可能是买入机会 ✅
```

**分位值区间：**
| 分位值 | 估值状态 | 投资建议 |
|--------|----------|----------|
| 0-20% | 严重低估 | 强烈买入 |
| 20-40% | 低估 | 买入 |
| 40-60% | 合理 | 持有 |
| 60-80% | 偏高 | 谨慎 |
| 80-100% | 高估 | 考虑卖出 |

**为什么分位值重要？**
```
PE绝对值：6.5倍
→ 这个数字是高还是低？

分位值：15%
→ 历史上只有15%的时间更低
→ 当前是历史低位 ✅

结论：分位值提供了相对位置
```

---

### 2. P50（中位数）

**定义：** 所有历史数据按大小排序后，位于中间位置的值

```
P50 = 中位数
     = 50%的数据小于它，50%的数据大于它
```

**示例（招商银行PE）：**
```
近10年PE数据排序：
4.7, 5.2, 6.1, 6.5, 7.0, 7.8, 8.3, 9.1, 10.2, 12.5
                      ↑
                    P50 = 7.0

含义：一半时间PE低于7.0，一半时间高于7.0
```

**为什么用中位数而不是平均值？**
```
平均值：容易受极端值影响
中位数：更稳定，更代表性

示例：
正常年份PE：5, 6, 7, 8, 9
极端年份PE：5, 6, 7, 8, 20（牛市泡沫）

平均值 = 9.2（被20拉高）❌
中位数 = 7（反映真实水平）✅
```

---

### 3. 其他常用分位数

| 符号 | 含义 | 说明 |
|------|------|------|
| **P10** | 10%分位数 | 10%的时间低于此值 |
| **P25** | 25%分位数 | 下四分位数，低估区边界 |
| **P50** | 50%分位数 | 中位数 |
| **P75** | 75%分位数 | 上四分位数，高估区边界 |
| **P90** | 90%分位数 | 90%的时间低于此值 |

**应用示例：**
```
招商银行PE估值区间：

P10 = 5.2倍  → 低于5.2严重低估
P25 = 6.1倍  → 6.1以下是低估区
P50 = 7.0倍  → 中位数，合理估值
P75 = 9.1倍  → 9.1以上是高估区
P90 = 10.2倍 → 高于10.2严重高估

当前PE = 6.5倍
→ 在P25和P50之间，合理偏低 ✅
```

## 五、估值区间判断

### 方法1：标准差法（我们使用的）

```
均值 = μ
低估值线 = μ - 1σ
高估值线 = μ + 1σ

示例（招行）：
均值 = 8.53
标准差 = 2.19
低估值线 = 8.53 - 2.19 = 6.34
高估值线 = 8.53 + 2.19 = 10.71

当前PE = 6.5
→ 接近低估值线 ✅
```

### 方法2：分位数法（雪球等平台）

```
分位值 < 20% → 低估
分位值 20-40% → 偏低
分位值 40-60% → 合理
分位值 60-80% → 偏高
分位值 > 80% → 高估
```

### 方法3：极值法

```
历史最低PE = 4.66
历史最高PE = 14.43

判断：
当前PE 接近历史最低 → 低估
当前PE 接近历史最高 → 高估
```

## 六、实战案例：招商银行

### 数据对比

| 类型 | PE值 | 说明 |
|------|------|------|
| 静态PE | 6.83 | 用2023年EPS |
| TTM PE | 6.50 | 用最近4季度EPS ✅ |
| 预期PE | 6.24 | 用2025年预测EPS |

### 估值判断

**方法1：分位值**
```
当前PE = 6.5
历史分位值 = 15%

→ 处于历史15%分位
→ 严重低估 ✅
```

**方法2：标准差**
```
当前PE = 6.5
低估值线 = 6.34

→ 接近低估线 ✅
```

**方法3：P50对比**
```
当前PE = 6.5
P50（中位数）= 7.0

→ 低于中位数 ✅
```

**综合判断：**
```
三种方法一致显示低估
→ 可靠性高 ✅
```

## 七、市盈率的优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 简单直观 | 容易理解和计算 |
| 应用广泛 | 所有盈利公司可用 |
| 横向可比 | 同行业公司对比 |
| 纵向可比 | 历史估值分位 |
| 国际通用 | 全球投资者使用 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 忽略成长 | 不考虑未来增长 |
| 周期股失效 | 高峰期低PE可能是陷阱 |
| 亏损不能用 | EPS为负无法计算 |
| 会计影响 | EPS可能被调节 |
| 行业差异 | 不同行业PE标准不同 |

---

## 八、适用场景

### ✅ 最适合

| 类型 | 原因 |
|------|------|
| 盈利稳定公司 | EPS可预测 |
| 消费龙头 | 业绩稳定增长 |
| 金融股 | 银行、保险等 |
| 成熟期公司 | 增长放缓但稳定 |

### ❌ 不太适合

| 类型 | 原因 |
|------|------|
| 亏损公司 | EPS为负 |
| 周期股 | 盈利波动大 |
| 高成长股 | PE高但合理 |
| 扭亏公司 | 剧烈变化 |

## 九、PE使用技巧

### 1. 结合分位值

```
PE绝对值：6.5倍
→ 不知道是高是低

分位值：15%
→ 历史低位 ✅

→ 两者结合更准确
```

### 2. 多方法验证

```
分位值法：低估 ✅
标准差法：低估 ✅
P50对比：偏低 ✅

多方法一致 → 更可靠
```

### 3. 行业对比

```
招行PE：6.5
行业PE：5.5

→ 招行有溢价，但质地更好
→ 合理 ✅
```

### 4. 关注趋势

```
PE上升：可能是股价涨或盈利跌
PE下降：可能是股价跌或盈利涨

需要：
- 分析PE变化原因
- 判断是否可持续
```

## 十、常见误区

### 误区1：PE越低越好

```
错误：PE最低就是最好的

可能原因：
- 行业景气到顶
- 公司出现危机
- 市场崩溃前夕

需要：
- 分析低PE原因
- 结合其他指标
- 判断是否可持续
```

### 误区2：高PE一定贵

```
错误：PE高就不能买

例外情况：
- 高成长公司（PEG合理）
- 业绩暂时低谷
- 行业特殊（科技）

需要：
- 看PEG
- 分析成长性
- 判断增长可持续
```

### 误区3：同行业PE对比就够

```
错误：同一行业PE越低越好

问题：
- 质量差异
- 成长差异
- 风险差异

正确：
- 优质公司可以有溢价
- 差公司PE低可能是陷阱
```

### 误区4：历史PE分位足够

```
错误：看历史分位就够了

问题：
- 公司基本面变化
- 行业环境变化
- 市场环境变化

需要：
- 结合基本面分析
- 动态调整标准
```

## 十二、总结

### PE法的核心价值

```
1. 简单直观，易于理解
2. 应用广泛，全球通用
3. 横向可比，纵向对比
4. TTM PE及时准确
5. 分位值提供位置感
```

### 正确使用姿势

```
✅ 使用TTM PE
✅ 关注分位值
✅ 多方法验证
✅ 结合基本面
✅ 行业对比分析
✅ 动态跟踪

❌ 只看绝对值
❌ 忽视分位值
❌ 单一方法判断
❌ 脱离基本面
❌ 忽视行业特性
❌ 静态看待
```

### 记住

```
PE是工具，不是答案
分位值提供位置
TTM保证时效
综合分析最重要

低PE不等于好投资
高PE不等于坏投资
理解背后的逻辑
比记住数字更重要
```
