# DCF估值方法的核心理解

## 一句话解释

**DCF就是：假设公司未来5-10年能赚多少钱，通过数学模型倒推出今天它值多少钱。**

---

## 核心逻辑

```
DCF = 倒推法

现在的合理价格
= 未来所有能赚到的钱
- 折算到今天的价值
```

---

## 简单例子：投资一家小店

### 问题：这家店值多少钱？

**已知条件：**
- 未来5年每年预期利润：10万、11万、12万、13万、14万
- 5年后可以转让：100万
- 你要求的回报率：10%

**DCF计算（倒推）：**

```
第1年现值：10万 ÷ 1.1 = 9.09万
第2年现值：11万 ÷ 1.1² = 9.09万
第3年现值：12万 ÷ 1.1³ = 9.02万
第4年现值：13万 ÷ 1.1⁴ = 8.87万
第5年现值：14万 ÷ 1.1⁵ = 8.69万
转让现值：100万 ÷ 1.1⁵ = 62.09万

─────────────────────────────

合理价格 = 9.09 + 9.09 + 9.02 + 8.87 + 8.69 + 62.09
        = 106.85万
```

**投资判断：**
```
如果卖价 < 80万  → 便宜，买入 ✅
如果卖价 = 100万 → 合理
如果卖价 > 130万 → 贵了，不买
```

---

## DCF的本质问题

DCF回答的是：

```
"这家公司未来能赚这么多钱，
今天我应该花多少钱买它才划算？"
```

**而不是：**
```
"这家公司过去赚了多少钱，
所以它值这个价"
```

---

## 实战案例：招商银行

### 第一步：预测未来赚钱能力

```
假设未来5年净利润：
2025年：1635亿
2026年：1782亿
2027年：1942亿
2028年：2117亿
2029年：2308亿

5年后公司价值：25000亿（按5倍PE估算）
```

### 第二步：折算到今天

```
折现率（要求回报）：10%

现值计算：
2025年：1635 ÷ 1.1 = 1486亿
2026年：1782 ÷ 1.1² = 1472亿
2027年：1942 ÷ 1.1³ = 1459亿
2028年：2117 ÷ 1.1⁴ = 1446亿
2029年：2308 ÷ 1.1⁵ = 1433亿
终值：25000 ÷ 1.1⁵ = 15523亿

─────────────────────────────

公司总价值 = 22819亿
减去净债务：3000亿
股东权益价值 = 19819亿
每股价值 = 19819 ÷ 257亿股 = 77元
```

### 第三步：判断现在是否便宜

```
理论价值：77元
当前股价：38.67元

安全边际 = (77 - 38.67) ÷ 77 = 50%

→ 便宜！买入 ✅
```

---

## 关键假设的影响

DCF结果高度依赖假设，不同假设得到不同结果：

### 乐观情况

```
增长率：15%（更高）
折现率：8%（更低）

→ 理论价值可能 = 120元
```

### 保守情况（常用）

```
增长率：8-10%
折现率：10%

→ 理论价值 = 77元
```

### 悲观情况

```
增长率：3-5%（经济放缓）
折现率：12%（风险更高）

→ 理论价值可能 = 50元
```

### 结论

```
DCF算出的是一个区间：50-120元

我们用保守假设：77元
当前股价：38.67元

即使最悲观情况(50元)，
也比现在股价贵30%
→ 安全边际足够 ✅
```

---

## DCF与其他估值方法的区别

| 方法 | 逻辑 | 关注点 |
|------|------|--------|
| **DCF** | 看未来赚钱能力 → 倒推现在值多少钱 | 未来现金流 |
| **PE** | 看过去盈利 → 多少年回本 | 历史盈利 |
| **PB** | 看资产价值 → 股价是否低于净资产 | 资产价值 |
| **PEG** | 看成长速度 → 估值是否匹配成长 | 成长性 |
| **股息率** | 看分红回报 → 现金流收益 | 分红 |

## 理解DCF的关键点

### 1. 为什么需要折现？

```
今天的100元 > 明年的100元

原因：
- 可以投资获利
- 通货膨胀
- 风险补偿

折现就是把未来的钱
换算成今天的价值
```

### 2. 折现率是多少？

```
折现率 = 你要求的投资回报率

通常：
- 银行/稳定公司：8-10%
- 成长股：10-12%
- 高风险公司：12-15%

折现率越高，
算出来的价值越低（越保守）
```

### 3. 预测多久？

```
通常预测5-10年

原因：
- 太短：看不清楚价值
- 太长：预测不准确

重点：
- 前5年详细预测
- 5年后用"永续增长率"简化
```

### 4. 永续增长率是多少？

```
永续增长 = 长期可持续增长率

通常：2-4%

原因：
- 不能超过GDP增长
- 不能一直高增长
- 接近通胀水平

保守估计用2-3%
```

---

## DCF的局限性

### 1. 假设敏感

```
小变化 → 大影响

折现率从10% → 11%
估值可能下降20%
```

### 2. 预测困难

```
谁能准确预测10年后？

- 行业变化
- 竞争加剧
- 技术颠覆
- 政策变化
```

### 3. 不适用情况

```
❌ 亏损公司（没有正现金流）
❌ 周期股（现金流波动大）
❌ 初创公司（无历史数据）
❌ 业务复杂（难以预测）
```

---

## 如何正确使用DCF

### 1. 保守假设

```
增长率：用历史下限
折现率：用高一点
永续增长：保守估计

宁要模糊的正确
不要精确的错误
```

### 2. 多方法验证

```
DCF估算内在价值
PE/PB对比估值
PEG结合成长性
股息率验证

多方法一致 → 更可靠
```

### 3. 留足安全边际

```
巴菲特原则：

买入价格 < 内在价值 × 80%

即：安全边际 > 20%

示例：
DCF内在价值 = 77元
最高买入价 = 77 × 80% = 62元
当前股价 = 38.67元
安全边际 = 50% ✅
```

### 4. 动态跟踪

```
DCF不是一次性的：

每季财报后更新
重大事项时调整
行业变化时修正
```

## 总结

### DCF回答的问题

```
"如果这家公司未来5-10年
每年赚X亿，最后值Y亿，
我今天应该花多少钱买它？"
```

### DCF的核心

```
未来赚钱能力 → 倒推 → 今天合理价格
```

### 记住

```
DCF是思维方式，不是计算工具
重点在逻辑，不在精确数字
保守假设 + 安全边际 = 成功投资
```

### 巴菲特的话

```
"DCF是评估一家公司价值的
唯一正确方法。"

但要记住：
"宁要模糊的正确，
不要精确的错误。"
```
