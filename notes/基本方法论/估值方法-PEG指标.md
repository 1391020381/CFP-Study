# PEG指标（市盈率增长比）详解

## 一、定义与公式

```
PEG = PE ÷ G

其中：
PE = 市盈率（通常用TTM）
G  = 预期净利润增长率（%）
```

**举例说明：**
```
PE = 20
预期增长率 = 20%
PEG = 20 ÷ 20 = 1
```

---

## 二、原理解释

### 谁提出的？

**彼得·林奇（Peter Lynch）**
- 著名基金经理
- 富达麦哲伦基金管理人
- 13年年化收益率29%

### 核心思想

```
PE只看估值，不看成长
PEG结合估值和成长

原理：
- 高PE公司如果成长快 → 可能合理
- 低PE公司如果不成长 → 不一定便宜
- PEG = 1 → 估值与成长匹配
```

### PEG代表什么？

| PEG值 | 含义 | 解释 |
|-------|------|------|
| PEG < 0.5 | 严重低估 | 成长快但估值低 |
| 0.5 ≤ PEG < 1 | 低估 | 估值低于成长速度 |
| PEG = 1 | 合理 | 估值与成长匹配 |
| 1 < PEG ≤ 1.5 | 略高 | 估值略高于成长 |
| 1.5 < PEG ≤ 2 | 偏高 | 估值明显高于成长 |
| PEG > 2 | 高估 | 估值过高或成长不足 |

---

## 三、判断标准

### PEG估值区间（彼得·林奇标准）

| PEG范围 | 估值状态 | 投资建议 |
|---------|----------|----------|
| PEG < 0.5 | 严重低估 | 强烈买入 |
| 0.5 ≤ PEG < 0.8 | 低估 | 买入 |
| 0.8 ≤ PEG < 1.2 | 合理 | 持有/买入 |
| 1.2 ≤ PEG < 1.5 | 略高 | 谨慎持有 |
| 1.5 ≤ PEG < 2 | 偏高 | 考虑卖出 |
| PEG ≥ 2 | 高估 | 卖出/回避 |

### 不同行业的PEG标准

| 行业 | 合理PEG | 原因 |
|------|---------|------|
| 成长股 | 1-1.5 | 成长性溢价 |
| 稳定消费 | 1-1.2 | 稳定增长 |
| 银行 | 0.8-1.2 | 成长较慢 |
| 科技 | 1.2-1.8 | 高成长预期 |
| 周期股 | 0.6-1 | 成长不稳定 |

## 四、实战案例：招商银行

### 基础数据（2026年）

```
招商银行(600036)：
- 当前PE：6.5
- 2024年净利润：约1500亿
- 2023年净利润：约1400亿
- 实际增长率：(1500-1400) ÷ 1400 = 7.1%
- 预期增长率（分析师一致预期）：约8%
```

### 计算PEG

**方法1：用历史增长率**

```
PE = 6.5
历史增长率 = 7.1%

PEG = 6.5 ÷ 7.1 = 0.92

判断：0.8 < 0.92 < 1.2 → 合理偏低 ✅
```

**方法2：用预期增长率**

```
PE = 6.5
预期增长率 = 8%

PEG = 6.5 ÷ 8 = 0.81

判断：0.8 < 0.81 < 1.2 → 合理偏低 ✅
```

### 估值结论

```
招商银行 PEG = 0.81-0.92

银行股标准（0.8-1.2）：
- 处于合理区间，略偏低
- 结合PE=6.5（低）
- 综合判断：低估 ✅
```

### 历史对比

| 年份 | PE | 增长率 | PEG | 判断 |
|------|-----|--------|-----|------|
| 2020 | 9.5 | 5% | 1.90 | 偏高 |
| 2021 | 12.0 | 20% | 0.60 | 低估 ✅ |
| 2022 | 6.5 | 10% | 0.65 | 低估 ✅ |
| 2023 | 5.5 | 5% | 1.10 | 合理 |
| 2024 | 8.3 | 7% | 1.19 | 合理 |
| 2026 | 6.5 | 8% | 0.81 | 合理偏低 ✅ |

## 五、对比不同公司

### 银行股对比（2026年数据）

| 公司 | PE | 预期增长率 | PEG | 评价 |
|------|-----|-----------|-----|------|
| 招商银行 | 6.5 | 8% | 0.81 | 合理偏低 ✅ |
| 工商银行 | 5.0 | 4% | 1.25 | 略高 |
| 建设银行 | 5.2 | 5% | 1.04 | 合理 |
| 农业银行 | 4.8 | 3% | 1.60 | 偏高 |
| 平安银行 | 7.0 | 12% | 0.58 | 低估 ✅ |

**分析：**
- 招行和平安PEG较低，更具吸引力
- 农行PE虽低但成长慢，PEG反而不便宜
- 工行PE适中但成长慢，PEG略高

### 科技股对比

| 公司 | PE | 预期增长率 | PEG | 评价 |
|------|-----|-----------|-----|------|
| 茅台 | 25 | 15% | 1.67 | 偏高 |
| 宁德时代 | 30 | 25% | 1.20 | 合理（科技股） |
| 比亚迪 | 25 | 20% | 1.25 | 合理 |
| 立讯精密 | 20 | 15% | 1.33 | 略高 |
| 海康威视 | 18 | 10% | 1.80 | 偏高 |

**分析：**
- 科技股PEG容忍度更高（1.2-1.8）
- 宁德时代、比亚迪PEG合理
- 茅台虽好但PEG偏高

## 六、为什么需要PEG？

### PE的局限性

```
问题1：高PE一定是贵吗？
公司A：PE=50，增长率50%
公司B：PE=20，增长率5%

单看PE：A贵B便宜
看PEG：
A: PEG = 50÷50 = 1 → 合理
B: PEG = 20÷5 = 4 → 高估

结论：A比B更值得投资！
```

```
问题2：低PE一定是便宜吗？
银行股：PE=5，增长率3%
科技股：PE=30，增长率30%

单看PE：银行便宜
看PEG：
银行: PEG = 5÷3 = 1.67 → 偏高
科技: PEG = 30÷30 = 1 → 合理

结论：科技股更合理！
```

### PEG的优势

| 优势 | 说明 |
|------|------|
| 结合成长 | 不只看估值，还看成长 |
| 跨行业比较 | 可比较不同行业公司 |
| 动态估值 | 考虑未来增长预期 |
| 彼得·林奇背书 | 大师使用的方法 |
| 简单实用 | 计算简单，易于理解 |

## 七、PEG法的优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 考虑成长性 | 弥补PE不足 |
| 跨行业可比 | 不同行业可比较 |
| 简单直观 | 易于理解计算 |
| 动态调整 | 随预期变化 |
| 大师认可 | 彼得·林奇推崇 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 增长率难预测 | 预期可能不准 |
| 不适用负增长 | 亏损公司无法用 |
| 忽略质量 | 只看速度不看质量 |
| 周期股失效 | 周期高峰PEG低但陷阱 |
| 线性假设 | 假设增长持续 |
| 失效于成长拐点 | 高成长放缓时PEG失真 |

## 八、适用场景

### ✅ 最适合

| 类型 | 原因 |
|------|------|
| 成长股 | 需要结合成长性评估 |
| 科技股 | 高PE高成长，PEG更有效 |
| 消费龙头 | 稳定增长的优质公司 |
| 医药股 | 持续成长的行业 |
| 新能源 | 高成长新兴行业 |

### ❌ 不太适合

| 类型 | 原因 |
|------|------|
| 周期股 | 盈利波动大，增长率不稳定 |
| 亏损公司 | 负增长无法计算PEG |
| 扭亏公司 | 增长率从负到正失真 |
| 成熟期公司 | 增长慢，PE更有用 |
| ST公司 | 财务数据异常 |

## 九、进阶应用

### 1. 动态PEG

```
动态PEG = 当前PE ÷ 未来3年平均增长率

优势：
- 更平滑
- 考虑增长趋势
- 避免单年异常

示例：
招行未来3年预期增长率：7%、8%、8%
平均 = 7.7%

动态PEG = 6.5 ÷ 7.7 = 0.84
```

### 2. PEG趋势分析

```
关注PEG变化趋势：

优秀公司：
PEG持续保持在1左右
或逐步下降

危险信号：
PEG持续上升
超过2甚至更高
```

### 3. PEG + ROE 组合

```
双重验证：

标准：
- PEG < 1
- ROE > 15%

示例：
宁德时代：
- PEG = 1.2 ✅
- ROE = 22% ✅
→ 优质成长股

某公司：
- PEG = 0.8 ✅
- ROE = 5% ❌
→ 盈利质量差，不是真便宜
```

### 4. 行业PEG比较

```
行业内PEG排序：

银行业：
- 招商 0.81 ✅ 最好
- 平安 0.58 ✅ 更好
- 建行 1.04 ✅ 合理
- 农行 1.60 ❌ 偏高

选择：
行业PEG最低的1-2家
```

## 十、常见误区与陷阱

### 误区1：PEG越低越好

```
错误：PEG=0.2一定很好

可能原因：
- 增长率被高估
- PE临时低（业绩暴雷前）
- 增长不可持续

需要：
- 分析增长可持续性
- 确认PE合理性
- 研究行业趋势
```

### 误区2：高PEG一定不能买

```
错误：PEG=2一定不能买

例外情况：
- 真正的高成长（初期）
- 短期业绩下滑
- 行业景气低点
- 投资期长

需要：
- 确认成长真实性
- 判断增长持续性
- 评估估值修复空间
```

### 误区3：用历史增长率

```
问题：历史 ≠ 未来

正确做法：
- 参考历史增长率
- 分析行业趋势
- 看分析师预期
- 做保守预测
```

### 误区4：PEG=1是买入点

```
错误：PEG=1就买入

需要考虑：
- 行业PEG标准不同
- 公司质量差异
- 增长确定性
- 估值位置

正确：
PEG < 行业平均
且质量好
才是好买点
```

### 误区5：周期股用PEG

```
问题：周期股PEG失效

周期股特征：
- 高峰期：利润暴增，PEG很低（陷阱）
- 低谷期：利润下滑，PEG很高（机会）

正确做法：
周期股用PB、PE分位数
不要用PEG
```

## 十一、实战策略

### 策略1：PEG选股法

```
步骤1：筛选低PEG公司
- PEG < 1
- 增长率 > 10%
- PE < 行业平均

步骤2：质量验证
- ROE > 12%
- 负债率合理
- 现金流良好
- 行业龙头

步骤3：综合判断
- 技术面
- 市场环境
- 估值位置
```

### 策略2：PEG拐点法

```
寻找PEG改善的公司：

特征：
- PEG从高位下降
- 增长率提升或PE下降
- 业绩拐点出现

买入时机：
- PEG降到1以下
- 业绩确认改善
- 技术面配合
```

### 策略3：PEG + PE双低

```
双重低估标准：
- PE < 历史均值
- PEG < 1

案例：2023年招行
- PE = 5.5（历史低位）
- PEG = 1.1（合理偏低）
→ 黄金买点 ✅
```

## 十二、注意事项

### 1. 增长率预测难点

```
增长率预测的不确定性：
- 分析师预期可能偏差大
- 行业变化快
- 公司自身波动

建议：
- 用保守预期
- 参考多家机构
- 关注趋势而非绝对值
- 定期修正预期
```

### 2. 不同增长阶段

```
初创期：高增长高PEG → 可接受
成长期：稳定增长PEG→1合理
成熟期：低增长PEG→PE为主
衰退期：负增长PEG→不适用
```

### 3. 质量vs速度

```
优质低速 vs 低质高速

优质低速（招行）：
- PE=6.5，增长率=8%
- PEG=0.81，稳健

低质高速：
- PE=30，增长率=30%
- PEG=1，风险大

选择：优质低速更可靠
```

### 4. PEG时效性

```
PEG需要动态更新：

季度更新：
- 每季财报后重新计算
- 跟踪预期变化
- 调整投资策略

年度复盘：
- 检查PEG历史表现
- 验证预测准确性
- 优化预测方法
```
