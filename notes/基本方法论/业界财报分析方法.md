## 一、财报分析基础概念

### 1.1 三大财务报表

| 报表 | 核心问题 | 分析重点 |
|------|----------|----------|
| **资产负债表** | 企业有什么？欠多少？ | 资产结构、负债水平、股东权益 |
| **利润表** | 企业赚了多少钱？ | 收入、成本、利润、盈利能力 |
| **现金流量表** | 钱从哪来？去哪了？ | 经营、投资、筹资三类现金流 |

### 1.2 会计核算基础

```
┌─────────────────────────────────────────────────────────────┐
│  权责发生制 vs 收付实现制                                    │
└─────────────────────────────────────────────────────────────┘

权责发生制（Accrual Basis）
  → 收入：商品交付或服务提供时确认，不管是否收到钱
  → 费用：义务发生时确认，不管是否支付
  → 适用：利润表、资产负债表
  
收付实现制（Cash Basis）
  → 收入/费用：只有实际收到/支付现金时才确认
  → 适用：现金流量表
```

**关键差异**：
- **营业收入**包含应收账款（卖了但没收到钱）
- **经营现金流**只计实际现金流入
- 两者差异是分析"盈利质量"的关键

### 1.3 单季度 vs 报告期 vs TTM

| 统计维度 | 说明 | 优点 | 缺点 |
|----------|------|------|------|
| **单季度** | 单个季度数据 | 反映最新情况，季节性明显 | 波动大 |
| **报告期** | 年初累计数据 | 消除部分季节性 | 滞后性 |
| **TTM** | 最近4个季度之和 | 平滑波动，反映最新12个月 | 并非完整财年 |

## 二、四大能力分析

财报分析的核心是评估企业的"四大能力"：

### 2.1 盈利能力分析 → 企业赚钱能力如何？

**核心指标**：

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| **ROE** | 净利润 / 股东权益 | 股东回报率 | >15% |
| **ROA** | 净利润 / 总资产 | 资产回报率 | >8% |
| **毛利率** | (收入-成本) / 收入 | 产品竞争力 | 因行业而异 |
| **净利率** | 净利润 / 收入 | 盈利能力 | >10% |
| **EBITDA率** | EBITDA / 收入 | 经营效率 | >20% |

**实战应用**：
```
案例：某公司ROE分解
  ROE = 20% → 优秀
  
  深入分析：
  - 高毛利（产品有竞争力）
  - 高周转（运营效率高）
  - 适度杠杆（财务风险可控）
```

### 2.2 偿债能力分析 → 企业还得起债吗？

**核心指标**：

| 指标 | 公式 | 说明 | 健康标准 |
|------|------|------|----------|
| **流动比率** | 流动资产 / 流动负债 | 短期偿债能力 | >1.5 |
| **速动比率** | (流动资产-存货) / 流动负债 | 快速偿债能力 | >1.0 |
| **资产负债率** | 总负债 / 总资产 | 财务杠杆 | <60% |
| **利息保障倍数** | EBIT / 利息支出 | 偿息能力 | >3倍 |

**实战应用**：
```
警惕信号：
  • 流动比率 < 1 → 短期偿债压力大
  • 资产负债率 > 70% → 杠杆过高
  • 利息保障倍数 < 2 → 偿债风险
```

### 2.3 运营能力分析 → 资产使用效率如何？

**核心指标**：

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| **存货周转率** | 营业成本 / 存货 | 存货周转速度 | 因行业而异 |
| **应收账款周转率** | 收入 / 应收账款 | 回款速度 | 越高越好 |
| **总资产周转率** | 收入 / 总资产 | 资产使用效率 | >0.8 |

**周转天数转换**：
```
存货周转天数 = 365 / 存货周转率
应收账款周转天数 = 365 / 应收账款周转率
```

**实战应用**：
```
周转天数变化分析：
  • 存货周转天数↑ → 可能是库存积压
  • 应收账款周转天数↑ → 回款变慢，占用资金
  • 两者同时上升 → 经营风险加大
```

### 2.4 成长能力分析 → 企业在发展吗？

**核心指标**：

| 指标 | 公式 | 说明 | 优秀标准 |
|------|------|------|----------|
| **营收增长率** | (本期-上期)/上期 | 规模扩张 | >20%优秀 |
| **净利润增长率** | (本期-上期)/上期 | 利润增长 | >15%优秀 |
| **CAGR** | (期末/期初)^(1/n)-1 | 复合年增长率 | >15%优秀 |

**TTM增长率分析**：
```
TTM（Trailing Twelve Months）= 滚动年度
  → 平滑单季度波动
  → 更准确反映最新趋势
  
TTM增长率 = (当期TTM - 去年同期TTM) / 去年同期TTM
```

**实战案例**：
```
比亚迪(002594) 近5年成长性：
  • 营收CAGR: 33.7% → 🚀 超高成长
  • 现金流CAGR: 19.4% → 高速成长
  • 成长能力评级: ⭐⭐⭐⭐⭐
```

## 三、现金流分析

### 3.1 三大类现金流

```
┌─────────────────────────────────────────────────────────────┐
│                    经营活动现金流 (OCF)                     │
│  → 企业"造血"能力                                           │
│  → 主营业务带来的现金流入/流出                              │
│  → 正常企业应为持续正值                                     │
└─────────────────────────────────────────────────────────────┘
                    +
┌─────────────────────────────────────────────────────────────┐
│                    投资活动现金流 (ICF)                     │
│  → 企业"扩张"能力                                           │
│  → 购买固定资产、对外投资                                   │
│  → 成长期通常为负值                                         │
└─────────────────────────────────────────────────────────────┘
                    +
┌─────────────────────────────────────────────────────────────┐
│                    筹资活动现金流 (FCF)                     │
│  → 企业"输血"能力                                           │
│  → 借款、还款、分红、融资                                   │
│  → 成长期通常为正值（融资扩张）                             │
└─────────────────────────────────────────────────────────────┘
                    =
              现金净增加额
```

### 3.2 现金流结构分析

**健康企业的现金流结构**：

| 阶段 | OCF | ICF | FCF | 说明 |
|------|-----|-----|-----|------|
| **初创期** | - | - | + | 依靠融资生存 |
| **成长期** | + | - | + | 有造血能力，持续投资 |
| **成熟期** | + | +/- | - | 分红、回购 |
| **衰退期** | +/- | + | - | 出售资产还债 |

**警惕信号**：
```
  • OCF持续为负 → 主业不造血
  • OCF<净利润且差距大 → 盈利质量差
  • 依赖融资维持运营 → 资金链风险
```

### 3.3 盈利质量分析：现金流 vs 净利润

**核心公式**：
```
经营现金流 ≈ 净利润 + 折旧摊销 - 应收账款增加 - 存货增加 + 应付账款增加
```

**现金流/净利润比值分析**：

| 比值 | 评价 | 说明 |
|------|------|------|
| >1.2 | 优秀 | 现金流充裕，盈利质量高 |
| 0.8-1.2 | 良好 | 现金流正常 |
| 0.5-0.8 | 一般 | 需关注应收账款 |
| <0.5 | 差 | 盈利质量差，可能有风险 |

**实战案例：比亚迪现金流分析**
```
最近5年TTM平均：
  • 营业收入: 5357 亿元
  • 经营现金流: 1251 亿元
  • 现金流/收入比: 23.3%

为什么现金流比收入低？
  1️⃣ 应收账款增加 → 卖车给经销商，账期30-90天
  2️⃣ 存货增加 → 扩大生产，占用现金
  3️⃣ 预付款项 → 预付原材料款锁定产能
  4️⃣ 折旧摊销 → 正向因素但影响有限

行业对比：比亚迪23.3%处于汽车制造行业【中等偏上】水平
```

### 3.4 自由现金流（FCF）

**定义**：企业可自由分配的现金

```
自由现金流 = 经营现金流 - 资本支出(CapEx)

用途：
  • 分红
  • 回购股票
  • 偿还债务
  • 并购投资
```

**FCF分析要点**：
```
  • FCF持续为正 → 企业创造超额现金
  • FCF持续为负 → 需要持续融资
  • FCF > 净利润 → 现金流非常健康
```

## 四、杜邦分析法

### 4.1 ROE分解

```
                    净利润
      ROE = ─────────────
                  股东权益

                    净利润     收入     总资产
          = ─────────── × ───── × ────────
                    收入      总资产   股东权益

          = 净利率 × 总资产周转率 × 权益乘数
            (盈利能力)   (运营效率)     (财务杠杆)
```

### 4.2 杜邦分析实战

| 公司 | 净利率 | 周转率 | 权益乘数 | ROE | 分析 |
|------|--------|--------|----------|-----|------|
| A | 20% | 0.8 | 1.5 | 24% | 优质：高盈利+高效率 |
| B | 5% | 2.0 | 2.0 | 20% | 薄利多销+高杠杆 |
| C | 15% | 0.5 | 3.0 | 22.5% | 高杠杆驱动，风险高 |

**分析要点**：
```
  • 净利率↑ → 产品有定价权
  • 周转率↑ → 运营效率高
  • 权益乘数↑ → 杠杆高，风险大
  
改善ROE的途径：
  1. 提高毛利率（涨价或降本）
  2. 加快资产周转（提高效率）
  3. 适度增加杠杆（但风险↑）
```

## 五、趋势与对比分析

### 5.1 趋势分析

**目的**：观察多期变化，识别发展趋势

**方法**：
```
  • 绝对值趋势：观察指标绝对值变化
  • 环比增长：与上一期比较
  • 同比增长：与去年同期比较
  • 复合增长率(CAGR)：长期平均增长
```

**TTM趋势分析优势**：
```
  • 消除单季度波动
  • 反映最新经营状况
  • 更早发现趋势变化
```

### 5.2 同业对比分析

**对比维度**：
```
  • 与行业平均比
  • 与龙头企业比
  • 与竞争对手比
```

**关键点**：
```
  • 不同行业指标差异大
  • 关注行业特性
  • 分析差异原因
```

### 5.3 历史对比分析

**与自己比**：
```
  • 与过去3年、5年比较
  • 识别进步或退步
  • 发现周期性规律
```

## 六、估值分析方法

### 6.1 PE估值法

```
市盈率(PE) = 股价 / 每股收益(EPS)

  • PE低 → 估值便宜（或增长前景差）
  • PE高 → 估值贵（或增长前景好）
  
  • 合理PE范围：因行业而异
    - 成长行业：20-40倍
    - 稳定行业：10-20倍
    - 周期行业：波动大
```

### 6.2 PB估值法

```
市净率(PB) = 股价 / 每股净资产

  • 适用于：金融、地产等资产密集型行业
  • PB < 1 → 破净（可能被低估）
```

### 6.3 PEG估值法

```
PEG = PE / 增长率

  • PEG < 1 → 低估
  • PEG = 1 → 合理
  • PEG > 1 → 高估
  
  • 考虑了增长因素，比PE更合理
```

### 6.4 DCF估值法

```
现金流折现模型(DCF)
  → 预测未来自由现金流
  → 按WACC折现到现值
  → 计算企业内在价值
  
  • 理论最合理
  • 但参数假设敏感
  • 适合现金流稳定的企业
```

## 七、风险分析

### 7.1 财务风险预警

**红色警报**：
```
  • 连续2年经营现金流为负
  • 现金流/净利润 < 0.3
  • 流动比率 < 0.8
  • 资产负债率 > 80%
  • 应收账款增长率 > 营收增长率
  • 存货周转天数持续上升
```

### 7.2 Z-Score破产预测模型

```
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5

  X1 = 营运资金/总资产
  X2 = 留存收益/总资产
  X3 = EBIT/总资产
  X4 = 股票市值/总负债
  X5 = 销售收入/总资产

  Z > 2.99 → 安全区
  1.81 < Z < 2.99 → 灰色区
  Z < 1.81 → 破产区
```

### 7.3 财务造假识别

**常见手段**：
```
  • 虚增收入（虚构合同、关联交易）
  • 提前确认收入
  • 隐瞒成本费用
  • 资本化费用（把费用当资产）
  • 隐瞒债务（表外负债）
```

**识别信号**：
```
  • 现金流与净利润长期背离
  • 应收账款异常增长
  • 毛利率显著高于同行
  • 频繁更换审计机构
  • 高管频繁离职
```

## 八、实战案例：比亚迪分析

### 8.1 成长能力分析

| 指标 | 数值 | 评价 |
|------|------|------|
| 5年营收增长 | 327.5% | 🚀 超高成长 |
| 营收CAGR | 33.7% | 🚀 超高成长 |
| 成长能力评级 | ⭐⭐⭐⭐⭐ | 优秀 |

### 8.2 盈利质量分析

| 指标 | 数值 | 评价 |
|------|------|------|
| 平均现金流/收入比 | 23.3% | ✅ 优秀 |
| 行业对比 | 汽车制造15-30% | 中等偏上 |

### 8.3 现金流分析结论

```
✅ 正面因素：
  • 现金流/收入比23.3% > 20%，盈利质量好
  • 高速成长期需要大量运营资金投入
  • 应收账款增加说明销售旺盛

⚠️ 需关注：
  • 单季度比值波动大(4%-40%)，季节性明显
  • Q1淡季现金流仅占收入5-10%
  • 需关注应收账款周转情况
```

### 8.4 综合评价

| 维度 | 评级 | 说明 |
|------|------|------|
| 成长能力 | ⭐⭐⭐⭐⭐ | 营收CAGR 33.7% |
| 盈利质量 | ⭐⭐⭐⭐⭐ | 现金流/收入23.3% |
| 行业地位 | ⭐⭐⭐⭐ | 汽车制造行业领先 |
