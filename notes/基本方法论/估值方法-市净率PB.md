# 市净率法（PB）详解

## 一、定义与公式

```
市净率 PB = 股价 ÷ 每股净资产
         = 总市值 ÷ 净资产
```

**每股净资产（BVPS）= 净资产 ÷ 总股本**

---

## 二、原理解释

### 什么是净资产？

净资产 = 总资产 - 总负债
       = 股东权益
       = 所有者权益

**简单理解：** 如果公司今天清算，把所有资产变现、还清债务后，剩下的钱就是净资产，理论上应该分给股东。

### PB代表什么？

| PB值 | 含义 | 每股1元净资产对应股价 |
|------|------|---------------------|
| 0.5 | 股价是净资产的一半 | 0.5元 |
| 1.0 | 股价等于净资产 | 1元 |
| 2.0 | 股价是净资产的2倍 | 2元 |

---

## 三、判断标准

### 银行股估值区间（最常用）

| PB范围 | 估值状态 | 投资建议 |
|--------|----------|----------|
| PB < 0.6 | 严重低估 | 强烈买入 |
| 0.6 ≤ PB < 0.8 | 低估 | 买入 |
| 0.8 ≤ PB < 1.2 | 合理 | 持有 |
| 1.2 ≤ PB < 1.5 | 偏高 | 谨慎 |
| PB ≥ 1.5 | 高估 | 考虑卖出 |

### 其他行业

| 行业 | 低估PB | 合理PB | 高估PB |
|------|--------|--------|--------|
| 银行 | <0.8 | 0.8-1.2 | >1.2 |
| 保险 | <0.8 | 0.8-1.5 | >1.5 |
| 地产 | <0.6 | 0.6-1.0 | >1.0 |
| 制造业 | <1.5 | 1.5-3.0 | >3.0 |
| 科技 | <2.0 | 2.0-4.0 | >4.0 |

---

## 四、实战案例：招商银行

### 数据获取（2026年数据）

```
招商银行(600036) 2026-01-30：
- 股价：38.67元
- 总股本：约257亿股
- 净资产：约11000亿元
```

### 计算过程

**方法1：用每股净资产**

```
每股净资产 = 净资产 ÷ 总股本
           = 11000亿 ÷ 257亿
           = 42.8元/股

PB = 股价 ÷ 每股净资产
   = 38.67 ÷ 42.8
   = 0.90
```

**方法2：用总市值**

```
总市值 = 股价 × 总股本
      = 38.67 × 257亿
      = 9938亿元

PB = 总市值 ÷ 净资产
   = 9938亿 ÷ 11000亿
   = 0.90
```

### 估值判断

```
招商银行 PB = 0.90

银行股标准：
- 0.8 ≤ PB < 1.2 → 合理估值
- 当前0.90处于合理区间，略偏低
```

### 历史对比

| 时期 | 股价 | 每股净资产 | PB | 判断 |
|------|------|-----------|-----|------|
| 2022年低点 | ~30 | ~35 | 0.86 | 低估 |
| 2023年 | ~32 | ~38 | 0.84 | 低估 |
| 2024年 | ~35 | ~40 | 0.88 | 低估 |
| 2026年现在 | 38.67 | 42.8 | 0.90 | 合理偏低 |

## 五、为什么银行股特别看重PB？

### 1. 银行资产质量相对稳定

- 银行主要资产是贷款、债券等金融资产
- 这些资产有明确的账面价值
- 相比制造业，银行资产折旧少

### 2. 银行盈利波动小

- 银行业务相对稳定
- 净资产增长平稳
- PB能较好反映价值

### 3. 银行净资产真实性高

- 受严格监管
- 资产需要减值测试
- 财务数据相对可靠

### 4. 历史验证有效

- 银行股估值中枢相对固定
- PB在0.8-1.2之间波动
- 破净（PB<1）往往机会大于风险

## 六、PB法的优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 数据易得 | 资产负债表数据 |
| 相对稳定 | 净资产比利润稳定 |
| 适合金融 | 银行、保险等特别有效 |
| 安全边际 | 破净提供底线保护 |
| 简单直观 | 容易理解和计算 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 忽略盈利能力 | 资产≠盈利能力 |
| 不适合轻资产 | 科技、服务类公司PB很高 |
| 会计影响 | 折旧政策影响净资产 |
| 周期性弱 | 不能反映周期变化 |
| 账面价值 ≠ 市场价值 | 有些资产可能虚高 |

---

## 七、适用场景

### ✅ 最适合

| 行业/公司类型 | 原因 |
|---------------|------|
| 银行 | 资产质量稳定，监管严格 |
| 保险 | 净资产是核心指标 |
| 券商 | 资产负债表清晰 |
| 房地产 | 土地储备价值明确 |
| 周期股 | 盈利波动大，净资产更稳定 |
| 国企 | 资产质量相对可靠 |

### ❌ 不太适合

| 行业/公司类型 | 原因 |
|---------------|------|
| 科技公司 | 轻资产，无形资产多 |
| 互联网 | 用户价值无法在资产表体现 |
| 咨询服务 | 人力资本为主 |
| 品牌消费 | 品牌价值不在资产负债表 |
| 亏损公司 | 净资产可能为负 |

## 八、进阶应用

### 1. PB-ROE配对分析

```
ROE = 净利润 ÷ 净资产

原理：高ROE公司理应有更高PB

优秀公司：高ROE + 合理PB → 可能低估
差公司：低ROE + 高PB → 可能高估
```

**案例对比：**

| 公司 | ROE | PB | 判断 |
|------|-----|-----|------|
| 招商银行 | 12% | 0.9 | 合理 |
| 某小银行 | 8% | 1.2 | 偏高 |
| 宁德时代 | 20% | 4.0 | 可能合理（高成长） |

### 2. 历史分位数法

```
当前PB在历史的位置

示例：
招商银行近10年PB范围：0.7 - 1.4
当前PB = 0.9
分位数 = (0.9-0.7) ÷ (1.4-0.7) = 29%
→ 处于历史29%分位，偏低
```

### 3. 相对估值法

```
公司PB vs 行业平均PB

示例：
招商银行 PB = 0.9
银行业平均 PB = 0.65
→ 招行有溢价，但质地更好，合理
```

### 4. 破净投资法

```
破净 = PB < 1

逻辑：
- 股价 < 清算价值
- 市场过度悲观
- 风险：可能是"价值陷阱"

适用：
- 银行、地产等行业
- 需要确认资产质量
- 避免财务造假公司
```

## 九、注意事项

### 1. 净资产质量

```
问题净资产：
- 应收账款可能坏账
- 存货可能积压
- 商誉可能减值
- 无形资产可能虚高

需要检查：
- 资产减值准备
- 商誉占比
- 无形资产构成
- 现金占比
```

### 2. 会计政策差异

```
影响净资产的因素：
- 折旧年限
- 存货计价方法
- 研发费用资本化
- 公允价值计量

不同公司会计政策可能不同
需要横向对比时注意调整
```

### 3. 行业周期位置

```
周期性行业：
- 高峰期：净资产高，PB低，但可能是陷阱
- 低谷期：净资产低，PB高，但可能是机会

示例：券商、有色金属
```

### 4. 新股次新股

```
刚上市的公司：
- 融资后净资产大增
- PB看似很低
- 但可能发行价就高估
```

### 5. PB ≠ 买入信号

```
破净不等于必然上涨

可能是：
- 行业整体衰落
- 公司经营恶化
- 财务造假风险
- 市场长期低迷

需要：
- 综合分析基本面
- 结合其他指标
- 考虑市场环境
```

## 十、实战建议

### 买入条件（多指标验证）

```
1. PB < 行业平均
2. PB处于历史30%分位以下
3. ROE > 10%
4. 资产质量良好（坏账率低、减值充足）
5. 行业基本面稳定
6. 盈利能力稳定或改善
```

### 卖出条件

```
1. PB > 历史平均值 + 1个标准差
2. PB远超行业平均
3. 资产质量恶化
4. 行业景气度见顶
5. 出现更好投资机会
```

### 持有条件

```
1. PB在合理区间
2. ROE保持稳定
3. 业务持续增长
4. 分红稳定增长
```

---

## 十一、总结

### PB法的核心价值

```
1. 提供安全边际
2. 适合重资产行业
3. 银行股核心指标
4. 简单易懂易用
```

### 正确使用姿势

```
✅ 结合ROE分析
✅ 历史纵向对比
✅ 行业横向对比
✅ 关注资产质量
✅ 综合其他指标

❌ 单独使用
❌ 忽视行业特性
❌ 盲目抄底破净
❌ 忽视基本面
```

### 记住

```
PB是工具，不是答案
低PB不等于好投资
理解背后的逻辑
比记住数字更重要
```
