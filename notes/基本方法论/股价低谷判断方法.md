# 股价低谷判断方法与原理

## 一、市盈率相关方法

### 1. 历史分位数法

```
当前PE在历史估值的位置 = 小于当前PE的历史天数 ÷ 总天数

示例：
当前PE = 6.5
历史上有80%的天数PE > 6.5
→ 当前处于历史20%分位，低估
```

**原理：** 股票估值会均值回归，极端低位通常对应买入机会

---

### 2. 标准差法

```
均值 = μ
低估区间 = μ - 1σ 或 μ - 2σ

示例：
均值 = 8.5，标准差 = 2.2
低估线 = 8.5 - 2.2 = 6.3
当前PE = 6.5 < 6.3 → 低估
```

**原理：** 正态分布，约68%的数据在±1σ内，95%在±2σ内

---

### 3. 行业对比法

```
公司PE vs 行业PE

示例：
招行PE = 6.5
银行业PE = 5.5
→ 招行相对合理，非绝对低估
```

**原理：** 同行业公司估值具有可比性

---

## 二、市净率法（PB）

```
PB = 股价 ÷ 每股净资产

银行股常用：
PB < 0.6 → 严重低估
0.6 < PB < 0.8 → 低估
0.8 < PB < 1.2 → 合理
PB > 1.2 → 高估
```

**原理：**
- 净资产是公司的"底线价值"
- 银行资产质量稳定，PB有效
- 破净（PB<1）意味着股价低于清算价值

---

## 三、股息率法

```
股息率 = 每股股息 ÷ 股价

判断标准：
股息率 > 5% → 低估
股息率 > 3% → 合理偏低
股息率 < 2% → 可能高估
```

**原理：**
- 股息是实实在在的现金流回报
- 高股息率提供安全边际
- 分红稳定性反映公司质地

---

## 四、PEG指标（市盈率增长比）

```
PEG = PE ÷ 净利润增长率

判断标准：
PEG < 0.5 → 严重低估
0.5 < PEG < 1 → 低估
PEG = 1 → 合理
PEG > 1 → 高估
PEG > 2 → 严重高估

示例：
招行PE = 6.5
预期增长率 = 8%
PEG = 6.5 ÷ 8 = 0.81 → 相对低估
```

**原理：** 彼得·林奇推崇的指标，结合估值与成长性

---

## 五、技术指标

### 1. 移动平均线

```
价格偏离MA的程度

示例：
股价 = 30
MA250 = 35
偏离度 = (30-35)/35 = -14%
→ 低于长期均线约14%，可能超跌
```

**原理：** 长期均线代表持仓成本，大幅偏离会回归

### 2. RSI相对强弱指标

```
RSI < 30 → 超卖（可能低点）
RSI > 70 → 超买（可能高点）
```

**原理：** 价格波动有节奏，极端状态不可持续

---

## 六、内在价值法（DCF）

```
公司价值 = 未来现金流折现之和

低买条件：股价 < 内在价值的80%
```

**原理：** 企业的真实价值是未来创造的现金流

---

## 七、市场情绪指标

### 1. 换手率

```
换手率 < 1% → 无人问津，可能低点
换手率 > 10% → 过度活跃，需警惕
```

### 2. 融资融券

```
融资盘爆仓 → 恐慌性抛售，往往伴随底部
```

---

## 八、综合判断矩阵（以招商银行为例）

| 方法 | 招行当前状态 | 判断 |
|------|-------------|------|
| PE分位数 | ~20% | 低估 ✅ |
| PE vs 均值-1σ | 6.5 vs 6.3 | 接近低估 ✅ |
| PB | ~0.9 | 合理偏低 |
| 股息率 | ~5.5% | 高股息 ✅ |
| PEG | 0.8 | 相对低估 ✅ |
| vs MA250 | 略低于 | 偏离不大 |

**结论：** 多指标共振显示低估，可靠性更高

---

## 九、不同行业适用的方法

| 行业 | 主要方法 | 辅助方法 |
|------|----------|----------|
| 银行 | PB、PE、股息率 | PEG |
| 成长股 | PEG | PE分位数 |
| 周期股 | PB | 现金流 |
| 稳定消费 | PE、股息率 | PEG |
| 科技 | PEG、PS | 市销率 |

## 十、注意事项

### 1. 低估 ≠ 马上涨

- 可能长期低估（价值陷阱）
- 需要结合基本面分析
- 耐心等待均值回归

### 2. 特殊时期方法可能失效

- 利率环境变化
- 行业政策变化
- 公司基本面恶化
- 宏观经济危机

### 3. 量化+定性结合

```
量化指标给出信号 → 定性分析确认逻辑
```

- 研究公司基本面
- 理解行业趋势
- 判断市场情绪
- 评估风险因素

### 4. 动态跟踪

- 估值是相对的、动态的
- 定期更新数据
- 关注边际变化

---

## 十一、实战建议

### 买入时机判断

1. **多指标共振**：至少3个指标同时显示低估
2. **基本面配合**：公司经营正常，无重大利空
3. **分批建仓**：不要一次性买入
4. **预留安全边际**：低估后再打折扣买入

### 风险控制

1. **分散投资**：不要押注单一股票
2. **设置止损**：判断错误及时退出
3. **定期复盘**：检验判断逻辑
4. **持续学习**：更新估值方法
